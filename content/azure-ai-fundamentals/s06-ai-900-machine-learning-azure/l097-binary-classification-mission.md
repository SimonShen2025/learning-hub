## 中文短总结

实验目标说明：使用 Designer 构建二元分类模型。数据集：Adult Census Income（内置），~32000行/15列。目标：基于输入特征（age, work_class, education, occupation 等）预测收入是 ≤50K 还是 >50K。这是 supervised learning + binary classification（两个可能输出值）。使用内置数据集和组件，无需编码。

## 考试要点

- Binary Classification = 输出只有两个类别
- Supervised Learning：需要 features + labels
- 数据集：features（输入特征） + label（income ≤50K or >50K）
- 使用 Designer 的 no-code 方式
- Classification vs Regression 的区别

## English Short Summary

Lab mission: build a binary classification model using Designer. Dataset: Adult Census Income (built-in, ~32K rows, 15 columns). Goal: predict income ≤50K or >50K based on features (age, work_class, education, occupation, etc.). Supervised learning + binary classification (two possible output values). No-code approach using built-in datasets and components.
