## 中文短总结

ML Studio 导览。从 Workspace Overview → Launch Studio。核心组件：1) **Designer**（拖拽界面，无代码构建 ML 工作流）；2) **Notebooks**（Python 代码方式）；3) **Data**（数据集管理）；4) **Compute**（计算实例管理）；5) **Pipelines**（ML 流水线）。Designer 中可使用预构建数据集和组件（数据转换、模型评分/评估等）。是低代码/无代码 ML 平台。

## 考试要点

- ML Studio = Web 界面，低代码/无代码 ML 平台
- 从 Workspace → Launch Studio 进入
- Designer：拖拽界面，无需编码
- Notebooks：Python 编程方式
- 预构建组件：数据转换、模型训练/评分/评估
- 可使用内置示例数据集

## English Short Summary

ML Studio tour: launch from Workspace Overview. Key components: 1) Designer (drag-and-drop, no-code ML workflows), 2) Notebooks (Python coding), 3) Data (dataset management), 4) Compute (instance management), 5) Pipelines. Designer uses pre-built datasets and components (data transformation, model scoring/evaluation). Low-code/no-code ML platform.
