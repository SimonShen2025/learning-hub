## 中文短总结

为什么使用 ML？传统编程：写逻辑规则 → 代码膨胀复杂。ML：用数据教计算机执行任务，构建模型发现数据模式。实际用例：垃圾邮件检测、手写数字识别（不用写 26 种字母规则）、客户流失预测、推荐系统、欺诈检测、图像分类。核心转变：从"编写规则"到"从数据学习"。

## 考试要点

- 传统编程 = 人写规则/逻辑，代码随时间膨胀
- ML = 用数据训练模型，模型自动发现模式
- 适合规则难以穷举的复杂场景
- ML 模型 = 数学表示的文件

## English Short Summary

Why ML? Traditional programming writes rules → code bloats over time. ML teaches computers via data, building models that discover patterns. Use cases: spam detection, handwriting recognition, churn prediction, recommendations, fraud detection. Core shift: from "writing rules" to "learning from data." ML model = mathematical representation as files.
