## 中文短总结

ML 数据核心概念：Features（输入特征/列）= 用于训练的输入变量；Labels（标签）= 要预测的输出值。数据准备：识别数据源 → 确保数据可用（处理缺失值、异常值、重复数据）→ 特征选择（选择相关特征）→ 数据分割（训练集/验证集/测试集）。数据质量直接影响模型准确性。存储成本下降使海量数据可用于训练。

## 考试要点

- Features = 输入变量/列
- Labels = 要预测的输出值（监督学习需要）
- 数据分割：训练集 + 验证集 + 测试集
- 数据质量 → 模型质量
- 数据准备步骤：缺失值、异常值、重复数据处理
- 特征选择：选择对预测有贡献的特征

## English Short Summary

ML data concepts: Features = input variables/columns for training; Labels = output values to predict. Data preparation: identify sources → ensure usability (handle missing values, outliers, duplicates) → feature selection → data splitting (train/validation/test sets). Data quality directly impacts model accuracy.
