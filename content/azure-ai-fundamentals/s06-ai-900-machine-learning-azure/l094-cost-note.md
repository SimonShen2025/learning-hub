## 中文短总结

ML 实验费用说明：讲师全部练习花费约 $1.5 USD（主要是 VM 计算和 AKS 集群）。费用来源：Pipeline 运行的 VM 计算 + AKS 端点托管。成本控制：Subscriptions → Cost Management → Budgets → 设置月度预算（如 $20）→ 配置警报（如达到 70% 时邮件通知）。

## English Short Summary

ML lab costs: ~$1.5 USD for all exercises (VM compute for pipelines + AKS cluster for endpoints). Cost control: Subscriptions → Cost Management → Budgets → set monthly budget (e.g., $20) → configure alerts (e.g., email at 70% threshold).
