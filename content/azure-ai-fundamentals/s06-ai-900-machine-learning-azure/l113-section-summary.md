## 中文短总结

Section 6 总结。ML 算法类型：**Classification**（Supervised，分类到类别，Binary/Multi-class）→ 如预测收入类别；**Regression**（Supervised，预测数值）→ 如预测销售/房价；**Clustering**（Unsupervised，发现数据分组）→ 如客户分群。数据概念：Features（输入）、Labels（输出）、Feature Selection、Feature Engineering、Train/Test Split。Azure ML 工具：Designer（no-code）、Notebooks（Python）、AutoML（全自动）。

## 考试要点

- **Classification**（Supervised）：Binary / Multi-class
- **Regression**（Supervised）：预测数值
- **Clustering**（Unsupervised）：数据分组
- Features = 输入, Labels = 输出（Supervised 需要）
- Feature Selection + Feature Engineering
- Train/Test Split（70:30 或 80:20）
- Azure ML 工具：Designer / Notebooks / AutoML
- 评估：Confusion Matrix, Accuracy, Precision, Recall

## English Short Summary

Section 6 recap. ML types: Classification (Supervised, categorize data, Binary/Multi-class); Regression (Supervised, predict numerical values); Clustering (Unsupervised, discover groups). Data: Features (input), Labels (output), Feature Selection/Engineering, Train/Test Split. Azure ML tools: Designer (no-code), Notebooks (Python), AutoML (fully automated). Evaluation: Confusion Matrix, Accuracy, Precision, Recall.
