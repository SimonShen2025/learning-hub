## 中文短总结

创建 Azure ML Workspace 实操。步骤：All Resources → Create → 搜索 "Machine Learning" → 配置向导（Subscription、Resource Group、Workspace Name[需唯一]、Region）→ Review + Create。创建 Workspace 同时自动创建：Application Insights、Log Analytics、Key Vault、Storage Account（考试不考这些附属资源）。

## 考试要点

- ML Workspace = Azure ML 所有活动的容器
- 创建时自动附带：App Insights、Log Analytics、Key Vault、Storage Account
- Workspace name 需全局唯一
- 需要 Subscription + Resource Group

## English Short Summary

Create Azure ML Workspace: All Resources → Create → search "Machine Learning" → configure (Subscription, Resource Group, Workspace Name [must be unique], Region) → Review + Create. Auto-creates: Application Insights, Log Analytics, Key Vault, Storage Account (not exam-relevant).
