## 中文短总结

Azure Machine Learning 服务概述。两种方式创建模型：1) Python 编程（Notebooks，需要编程背景）；2) Designer（拖拽界面，无代码）。ML Studio 管理整个 ML 项目生命周期。Designer 允许非程序员通过拖放数据集和预构建组件来构建模型。下一步：创建 ML Workspace → 使用 Designer 创建模型。

## 考试要点

- Azure ML Service = 管理整个 ML 项目生命周期
- 两种方式：Python Notebooks（代码）vs Designer（无代码拖拽）
- Designer：适合非程序员
- ML Workspace：所有 ML 活动的容器
- ML Studio：Web 界面工作区

## English Short Summary

Azure Machine Learning service overview. Two approaches: 1) Python programming (Notebooks, requires coding skills); 2) Designer (drag-and-drop, no-code). ML Studio manages entire ML project lifecycle. Designer allows non-programmers to build models by dragging datasets and pre-built components. Next steps: create ML Workspace → use Designer.
