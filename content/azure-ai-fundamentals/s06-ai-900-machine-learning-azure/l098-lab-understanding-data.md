## 中文短总结

在 Designer 中创建新 Pipeline，使用内置 Adult Census Income 数据集。步骤：Designer → New Pipeline（classic pre-built components）→ 命名 → 从 Components > Sample Datasets 拖入 "Adult Census Income Binary Classification" 数据集到 Canvas。数据集包含收入分类信息（≤50K / >50K）。Pipeline = 一系列处理步骤的工作流。

## English Short Summary

Create new Pipeline in Designer using built-in Adult Census Income dataset. Steps: Designer → New Pipeline (classic pre-built) → name it → drag "Adult Census Income Binary Classification" from Sample Datasets onto canvas. Pipeline = workflow of processing steps. Dataset contains income classification (≤50K / >50K).
