## 中文短总结

三大 ML 类型：1) **Supervised Learning**（有标签数据训练，features + labels）— 分类/回归；2) **Unsupervised Learning**（无标签，模型自行发现模式）— 聚类（如按购买行为分组客户）；3) **Reinforcement Learning**（奖惩机制，智能体通过试错学习最优策略）— 自动驾驶、游戏 AI。

## 考试要点

- **Supervised**：有标签数据 → 分类(Classification) + 回归(Regression)
  - 分类：预测离散类别（如欺诈/非欺诈）
  - 回归：预测连续数值（如房价）
- **Unsupervised**：无标签数据 → 聚类(Clustering)
  - 模型自行发现数据模式
  - 如：客户分群
- **Reinforcement Learning**：奖惩反馈学习
  - 智能体在环境中试错
  - 如：自动驾驶、游戏 AI

## English Short Summary

Three ML types: 1) Supervised Learning (labeled data: features + labels) — Classification & Regression; 2) Unsupervised Learning (no labels, model discovers patterns) — Clustering (e.g., customer segmentation); 3) Reinforcement Learning (reward/punishment, agent learns through trial-and-error) — self-driving, game AI.
