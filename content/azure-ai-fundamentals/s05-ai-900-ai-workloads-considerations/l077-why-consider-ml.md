## 中文短总结

ML 应用场景：异常检测（银行欺诈、网络异常）、NLP（虚拟助手、聊天机器人）、图像/对象分类（医学扫描、自动驾驶）、预测与预报（股市、客户行为、设备故障）。核心区别：传统编程 = 人写逻辑规则；ML = 提供数据，系统自主学习创建逻辑。例：垃圾邮件检测——不需写规则识别每种垃圾邮件，只需喂入已标记数据让模型学习。

## 考试要点

- ML 使用场景：异常检测、NLP、CV、预测
- 传统编程 vs ML 的根本区别
- ML 优势：适合规则难以穷举的场景
- 数据是 ML 的核心驱动力

## English Short Summary

ML use cases: anomaly detection (fraud, network), NLP (assistants, chatbots), image classification (medical, self-driving), prediction (stocks, behavior, failures). Key difference: traditional programming = human writes rules; ML = provide data, system learns its own logic. Example: spam detection without explicit rules.
