## 中文短总结

AI 代码生成：可用 OpenAI/GitHub Copilot/Google Gemini/Claude 等工具生成代码。示例：提交详细 prompt（指定语言、功能需求、技术细节）→ 模型生成完整代码。关键：prompt 越具体→输出越好。Claude（Anthropic 公司）= 另一个 LLM 平台，有类似的免费/付费层级。代码生成不只是文本生成——是 GenAI 的重要应用场景。

## English Short Summary

AI code generation: use OpenAI/GitHub Copilot/Google Gemini/Claude to generate code. Example: submit detailed prompt (specify language, features, tech details) → model generates complete code. Key: more specific prompt → better output. Claude (Anthropic) = another LLM platform with similar free/paid tiers. Code generation is a major GenAI application.
