## 中文短总结

OpenAI Developer Playground：可选择不同模型（GPT-5.5, 5.4, 5.4-nano 等）、设置输出格式（Text/JSON）、调整 reasoning effort（推理努力程度，越高→思考越深→回复越慢但越好）、添加 tools、设置 developer message + user messages。Reasoning = 模型"思考"后再回答（chain of thought），而非直接给答案。

## 考试要点

- Playground = 开发者测试模型的交互界面
- 可选择不同模型版本
- **Reasoning effort**：模型思考深度（低=快速回答，高=深度推理）
- 输出格式：Text / JSON
- Developer message + User message
- Chain of thought = 模型内部推理过程

## English Short Summary

OpenAI Developer Playground: select different models (GPT-5.5, 5.4, nano), set output format (Text/JSON), adjust reasoning effort (higher → deeper thinking → slower but better), add tools, set developer message + user messages. Reasoning = model "thinks" before answering (chain of thought) rather than immediate response.
