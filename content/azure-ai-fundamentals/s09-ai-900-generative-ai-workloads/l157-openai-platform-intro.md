## 中文短总结

OpenAI Platform vs ChatGPT：ChatGPT = 消费者界面（对话）；OpenAI Platform = 开发者平台（API 调用、集成到应用）。开发者需要 API 访问将 AI 能力集成到自己的应用中。GPT 模型家族：GPT-4.1（旗舰）、O3/O4-mini（推理模型）、GPT-4.1-mini（快速低成本）等。Azure 也以类似方式提供 OpenAI 模型（Microsoft Foundry）。

## 考试要点

- ChatGPT = 消费者版本（对话界面）
- OpenAI Platform = 开发者版本（API 调用）
- 从应用调用模型 = 使用 API
- Azure Microsoft Foundry 也提供 OpenAI 模型
- 多种模型可选（旗舰/推理/轻量）

## English Short Summary

OpenAI Platform vs ChatGPT: ChatGPT = consumer interface (conversation); OpenAI Platform = developer platform (API calls, app integration). Developers need API access to integrate AI into their apps. GPT model family: GPT-4.1 (flagship), O3/O4-mini (reasoning), GPT-4.1-mini (fast/cheap). Azure offers same models via Microsoft Foundry.
