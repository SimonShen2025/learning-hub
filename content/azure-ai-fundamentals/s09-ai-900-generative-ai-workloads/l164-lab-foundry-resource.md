## 中文短总结

创建 Microsoft Foundry 资源（用于在 Azure 上使用 OpenAI 模型）。步骤：All Resources → Create → 搜索 "Microsoft Foundry" → 配置（Resource Group、Name[需唯一]、Region、Project Name）→ Review + Create。可选配置：Key Vault（密钥管理）、Application Insights（性能监控）——初学者不需要。创建完成后通过 Foundry Portal（ai.azure.com）访问。

## English Short Summary

Create Microsoft Foundry resource (to use OpenAI models on Azure). Steps: All Resources → Create → search "Microsoft Foundry" → configure (Resource Group, Name [unique], Region, Project Name) → Review + Create. Optional: Key Vault, Application Insights (not needed for beginners). Access via Foundry Portal (ai.azure.com) after creation.
