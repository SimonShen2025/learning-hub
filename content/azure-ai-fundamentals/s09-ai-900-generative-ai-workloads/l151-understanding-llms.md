## 中文短总结

LLM（大语言模型）= 驱动 GenAI 的核心。GPT = Generative Pre-trained Transformer。三个词含义：Generative（生成新内容）、Pre-trained（在海量数据上预训练）、Transformer（特定神经网络架构，理解上下文和关系）。"Large" = 在互联网/书籍/文章/代码等海量数据上训练。Neural Network = 互连神经元系统（输入层→隐藏层→输出层）。

## 考试要点

- **LLM** = Large Language Model，专为 NLP 设计的 ML 模型
- **GPT** = Generative Pre-trained Transformer
  - Generative：生成新内容
  - Pre-trained：已在海量数据上训练
  - Transformer：神经网络架构（理解上下文）
- **Neural Network**：输入层 → 隐藏层 → 输出层
- "Large"：训练数据量巨大（互联网规模）
- Token = 文本处理的基本单位

## English Short Summary

LLM (Large Language Model) powers GenAI. GPT = Generative Pre-trained Transformer: Generative (creates new content), Pre-trained (trained on massive data), Transformer (neural network architecture for understanding context/relationships). "Large" = trained on enormous internet-scale data. Neural Network = interconnected neurons (input → hidden → output layers). Token = basic unit of text processing.
