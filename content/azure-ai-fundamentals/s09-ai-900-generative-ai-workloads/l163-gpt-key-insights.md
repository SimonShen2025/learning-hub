## 中文短总结

GenAI 关键概念总结：1) GenAI = 创造新内容的范式转变；2) LLM 驱动 GenAI（OpenAI/Anthropic/Google）；3) **Prompt** = 用户输入；4) **Tokens** = 文本处理基本单位（prompt → tokens → transformer → response tokens）；5) **Transformer** = 核心架构（理解上下文后逐 token 生成响应）；6) **Multi-modal** = 模型接受多种输入（文本/图像/文件/音频）；7) ChatGPT = 消费者版，Developer Platform = API 版。

## 考试要点

- **Prompt** = 用户输入（开始对话）
- **Tokens** = 文本处理基本单位（定价依据）
- **Transformer** = 核心架构（理解上下文 + 生成）
- **Multi-modal** = 接受多种输入类型
- GenAI 公司：OpenAI, Anthropic, Google
- ChatGPT vs Developer Platform（消费者 vs 开发者）

## English Short Summary

GenAI key concepts review: 1) GenAI = paradigm shift for creating new content; 2) LLMs power GenAI (OpenAI/Anthropic/Google); 3) Prompt = user input; 4) Tokens = basic text processing units (pricing basis); 5) Transformer = core architecture (understands context, generates response token by token); 6) Multi-modal = accepts multiple input types (text/images/files/audio); 7) ChatGPT = consumer, Developer Platform = API.
