## 中文短总结

ChatGPT 更多功能：可上传文件（照片、代码文件等）作为 prompt 的一部分。多模态输入：文本 + 文件 + 图像。示例：上传 Python 代码 + 提问"解释代码做什么" → 获得逐行解释。其他功能：Deep Research、Web Search、Canvas、Image Generation。AI 作为学习工具：不懂代码→提交+提问→获得解释。

## English Short Summary

ChatGPT additional features: upload files (photos, code) as part of prompts. Multimodal input: text + files + images. Example: upload Python code + ask "explain what this does" → get line-by-line explanation. Other features: Deep Research, Web Search, Canvas, Image Generation. AI as learning tool: don't understand code → submit + ask → get explanation.
