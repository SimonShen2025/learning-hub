## 中文短总结

注册 OpenAI Developer Platform：需预充值 $5 USD 才能使用 API。定价模式：基于 **Tokens**（输入 tokens + 输出 tokens）。Token = 文本处理的基本单位。对话中交换的 tokens 越多→费用越高。$5 足够大量实验。考试不要求实际操作 OpenAI Platform，但需理解 token 定价概念。

## 考试要点

- OpenAI 平台定价：基于 Tokens（输入 + 输出）
- Token = 文本处理基本单位
- 不是按请求计费，是按 token 计费
- 需要预充值才能使用 API
- Azure OpenAI 也使用相同的 token 定价模式

## English Short Summary

Sign up for OpenAI Developer Platform: requires $5 USD prepayment to use APIs. Pricing model: based on **Tokens** (input tokens + output tokens). Token = basic unit of text processing. More tokens exchanged → higher cost. $5 sufficient for extensive experimentation. Exam requires understanding token pricing concept, not actual platform operation.
