## 中文短总结

System Message（Developer Message）：指导模型的行为/角色/风格。与 user prompt 一起发送。示例："你是Azure认证教练，以友好鼓励的方式回答，将复杂概念简化"。效果：模型遵循系统指令调整输出风格/长度/格式。System message 设置一次→应用于整个对话。用于：角色设定、输出格式要求、行为约束。

## 考试要点

- **System Message** = 指导模型行为的指令（Developer Message）
- 与 user prompt 一起发送
- 设置：角色/风格/格式/约束
- 设置一次 → 应用于整个对话
- Prompt Engineering 的核心技巧之一

## English Short Summary

System Message (Developer Message): instructs model behavior/role/style. Sent alongside user prompts. Example: "You are an Azure certification coach, respond in friendly encouraging manner, simplify complex topics." Effect: model follows system instructions for output style/length/format. Set once → applies to entire conversation. Used for: persona, output format, behavioral constraints.
