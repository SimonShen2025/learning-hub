## 中文短总结

安装 VS Code：从 code.visualstudio.com 下载 → 运行安装程序 → 接受协议 → 勾选 "Register Code as editor" + "Add to PATH" → Install → Launch。VS Code 是轻量级 IDE，用于编写 Python 代码调用 AI 模型。

## English Short Summary

Install VS Code: download from code.visualstudio.com → run installer → accept agreement → check "Register Code as editor" + "Add to PATH" → Install → Launch. VS Code is lightweight IDE for writing Python code to call AI models.
