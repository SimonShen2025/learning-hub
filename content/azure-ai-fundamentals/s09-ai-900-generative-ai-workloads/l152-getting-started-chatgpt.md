## 中文短总结

ChatGPT = OpenAI 的消费者版 AI 工具。免费版有限制（每日消息量/功能受限），付费版（~$20/月）解锁全部功能和更少限制。核心操作：发送 prompt → 模型生成响应。Prompt 是与模型对话的起点。模型通过大量数据训练，理解你的查询并生成相关响应。考试需了解：如何从 Azure（Microsoft Foundry）使用 OpenAI 模型。

## English Short Summary

ChatGPT = OpenAI's consumer AI tool. Free version has limitations (daily message caps, fewer features); paid ($~20/month) unlocks all features. Core operation: send prompt → model generates response. Prompt starts conversation with model. Model trained on massive data understands queries and generates responses. Exam focus: using OpenAI models from Azure (Microsoft Foundry).
