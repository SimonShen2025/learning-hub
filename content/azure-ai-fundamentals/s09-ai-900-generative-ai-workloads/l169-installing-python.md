## 中文短总结

安装 Python：从 python.org 下载最新版 → 勾选 "Add python.exe to PATH" + "Use admin permissions" → Install。验证：命令行运行 `python --version`。简单直接的安装过程。

## English Short Summary

Install Python: download latest from python.org → check "Add python.exe to PATH" + "Use admin permissions" → Install. Verify: run `python --version` in command prompt. Simple straightforward installation.
