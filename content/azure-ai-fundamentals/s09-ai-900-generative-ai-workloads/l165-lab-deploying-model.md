## 中文短总结

在 Microsoft Foundry 中部署模型。进入 Foundry Portal（ai.azure.com）→ Discover → Models → 可按能力/来源筛选（OpenAI, DeepSeek, Anthropic 等多厂商模型）→ 选择 GPT 模型 → Deploy。Foundry 的优势：不限于单一厂商，多供应商模型统一平台。新模型发布后几乎立即在 Foundry 可用。部署 = 在 Microsoft 管理的基础设施上运行模型。

## 考试要点

- Microsoft Foundry = 多供应商模型平台
- Discover → Models 查看所有可用模型
- 可按能力/来源筛选
- Deploy = 在 Azure 基础设施上部署模型
- 不限于 OpenAI（也有 DeepSeek, Anthropic 等）

## English Short Summary

Deploy model in Microsoft Foundry. Enter Foundry Portal (ai.azure.com) → Discover → Models → filter by capability/source (OpenAI, DeepSeek, Anthropic — multi-vendor) → select GPT model → Deploy. Foundry advantage: not tied to single vendor, unified platform for multiple providers. New models available almost immediately after release. Deploy = run model on Microsoft-managed infrastructure.
