## 中文短总结

Azure AI Language 是处理自然语言的云服务，现作为 Foundry Tools 的一部分。核心功能：Language Detection（识别文本语言）、Named Entity Recognition/NER（识别人名、地点、组织、日期）、Sentiment Analysis（正面/负面/中性情感判断）、PII Detection（个人可识别信息检测：姓名、邮箱、电话）、Key Phrase Extraction（提取文档关键短语）。无需从零构建模型，即用型能力。

## 中文长总结

### Azure AI Language 定义
- 提供 NLP 功能的云服务
- 理解和分析文本
- 现在作为 Foundry Tools 中的 Azure Language 呈现
- 早期有独立的 Language Studio

### 核心功能

| 功能 | 描述 | 用例 |
|------|------|------|
| Language Detection | 检测文本语言 | 多语言客服系统 |
| Named Entity Recognition (NER) | 识别实体：人名、地点、组织、日期 | 文档分析、新闻处理 |
| Sentiment Analysis | 判断正面/负面/中性 | 电商评论、社交媒体 |
| PII Detection | 检测个人信息：姓名、邮箱、电话 | 合规、数据保护 |
| Key Phrase Extraction | 提取文档关键短语 | 文档摘要、搜索优化 |

### 特点
- 即用型能力（ready-to-use），无需训练
- 直接传入文本，返回结构化结果
- 适用：客服系统、文档处理、合规搜索

## 考试要点

- NER 识别：人名、地点、组织、日期
- PII 识别：姓名、邮箱、电话等敏感信息

## English Short Summary

Azure AI Language is a cloud NLP service (now part of Foundry Tools). Key features: Language Detection, Named Entity Recognition (people, places, organizations, dates), Sentiment Analysis (positive/negative/neutral), PII Detection (names, emails, phones), Key Phrase Extraction. All prebuilt — no model training needed. Used for support systems, document processing, and compliance.
