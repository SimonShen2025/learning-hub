## 中文短总结

Azure AI Translator 服务：文本翻译和文档翻译。文本翻译可实时将文本从一种语言转换到另一种。用例：多语言聊天应用、网站/应用本地化、客服翻译。文档翻译可保留原始布局和结构。还可构建自定义翻译器。在 Foundry 中访问：Build → Models → AI Services → Azure Translator。操作简单：输入源文本 → 选择目标语言 → 翻译。

## 中文长总结

### Azure Translator 功能

| 功能 | 说明 |
|------|------|
| Text Translation | 实时文本翻译 |
| Document Translation | 保留布局的文档翻译 |
| Custom Translator | 自定义翻译模型 |

### 用例
- 多语言聊天应用
- 网站和应用本地化
- 客服翻译（客户来信 → 翻译理解 → 翻译响应回去）
- 全文档翻译（保留格式）

### 在 Foundry 中使用
- Build → Models → AI Services → Azure Translator
- 选择 Text 或 Document
- 输入源文本 → 选择目标语言 → 翻译

## 考试要点

- Azure Translator = AI 驱动的语言翻译服务
- 支持 Text Translation 和 Document Translation
- Document Translation 保留原始布局
- 可构建 Custom Translator
- 在 Foundry Tools 中作为 AI Service 提供

## English Short Summary

Azure AI Translator provides text translation (real-time between languages) and document translation (preserving layout/structure). Use cases: multilingual chat apps, website localization, customer support translation. Also supports custom translators. Access in Foundry: Build → Models → AI Services → Azure Translator.
