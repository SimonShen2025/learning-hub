## 中文短总结

Named Entity Recognition (NER)：从非结构化文本中自动识别和分类有意义的实体。实体类型：Person（人名）、Location（地点）、Organization（组织）、DateTime（日期）、Event（事件）、Quantity（数量）。示例：邮件中识别出人名、餐厅位置、周纪念日。应用：保险理赔处理（从描述中提取人/地/时信息）。API kind: "EntityRecognition"。

## 考试要点

- NER = 从文本中识别和分类实体
- 实体类型：Person, Location, Organization, DateTime, Event, Quantity
- 输入非结构化文本 → 输出结构化实体
- API kind: "EntityRecognition"
- 应用：保险理赔、文档处理、信息提取

## English Short Summary

Named Entity Recognition (NER): automatically identify and categorize meaningful entities from unstructured text. Entity types: Person, Location, Organization, DateTime, Event, Quantity. Example: email identifies names, restaurant location, anniversary date. Application: insurance claim processing (extract people/places/dates from descriptions). API kind: "EntityRecognition".
