## 中文短总结

知识库测试和部署：Test Knowledge Base → mini chat area → 提问 → 从知识库获取答案（显示来源和置信度分数）。测试满意后 → Deploy → 外部应用可调用知识库端点。可添加 "Chit Chat"（预制闲聊 Q&A bank，如 Professional/Friendly 风格）。完整流程：构建知识库 → 测试 → 部署 → 应用集成（chatbot 前端）。

## 考试要点

- 知识库测试：提问 → 查看答案 + 来源 + 置信度
- 部署后：应用通过端点调用知识库
- Chit Chat：预制闲聊问答库（可选添加）
- 完整流程：构建 → 测试 → 部署 → 集成

## English Short Summary

Knowledge base testing and deployment: Test → mini chat area → ask questions → get answers from KB (shows source + confidence score). After testing → Deploy → external apps call KB endpoint. Can add "Chit Chat" (pre-built conversational Q&A bank in Professional/Friendly styles). Full flow: build KB → test → deploy → integrate (chatbot frontend).
