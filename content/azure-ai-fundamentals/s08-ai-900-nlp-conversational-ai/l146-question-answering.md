## 中文短总结

Custom Question Answering：基于自有数据构建对话式 AI 应用。核心：从 FAQ/文档/手册中构建知识库 → 用户用自然语言提问 → 获取答案。功能：从 URL/文件自动提取 Q&A 对、follow-up prompts 引导对话、active learning 持续改善答案质量。前置需求：Azure AI Search 资源（云搜索引擎，存储和索引知识库内容）。

## 考试要点

- Custom Question Answering = 基于自有数据的 Q&A 系统
- 需要 Azure AI Search 资源
- 从 URL/文件自动提取 Q&A 对
- Follow-up prompts：引导式对话
- Active learning：基于实际使用改善答案
- 知识库 = Q&A pairs 的集合

## English Short Summary

Custom Question Answering: build conversational AI on your own data (FAQ, docs, manuals). Build knowledge base → users ask natural language questions → get answers. Features: auto-extract Q&A pairs from URLs/files, follow-up prompts for guided conversation, active learning for quality improvement. Requires Azure AI Search resource (cloud search engine for indexing knowledge base).
