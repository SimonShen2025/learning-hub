## 中文短总结

创建知识库实操：1) 部署 Azure AI Search 资源；2) Microsoft Foundry → Management Center → Connected Resources → 连接 AI Search；3) Fine Tuning → AI Service Fine Tuning → Custom Question Answering → Connect search service；4) Manage Sources → 添加 Q&A（从 URL/文件导入或手动添加）；5) Edit Knowledge Base → 添加/修改 Q&A 对。知识库 = 问答对集合。

## English Short Summary

Create knowledge base: 1) Deploy Azure AI Search resource; 2) Foundry → Management Center → Connected Resources → connect AI Search; 3) Fine Tuning → Custom Question Answering → connect; 4) Manage Sources → add Q&As (import from URLs/files or manual); 5) Edit Knowledge Base → add/modify Q&A pairs. Knowledge base = collection of Q&A pairs.
