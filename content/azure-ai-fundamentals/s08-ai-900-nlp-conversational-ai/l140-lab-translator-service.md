## 中文短总结

Azure AI Translator Service：跨 100+ 语言翻译文本和文档（实时或批量）。Microsoft 自己的产品（Word, Outlook）也使用此服务。在 Foundry → Translator Playground 中测试。双向翻译：德语客户消息→英语（支持团队理解）→回复翻译为德语。高度可扩展。也可通过 REST API 调用。

## 考试要点

- Azure Translator = 100+ 语言翻译
- 文本翻译 + 文档翻译
- 实时或批量处理
- Microsoft 自己产品也使用此服务
- 在 Translator Playground 中可测试
- REST API 可调用

## English Short Summary

Azure AI Translator Service: translate text and documents across 100+ languages (real-time or batch). Microsoft's own products (Word, Outlook) use this service. Test in Foundry → Translator Playground. Bidirectional: German customer message → English (support understands) → reply translated to German. Highly scalable. REST API available.
