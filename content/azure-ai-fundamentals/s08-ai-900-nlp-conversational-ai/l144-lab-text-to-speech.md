## 中文短总结

TTS 实操：Foundry → Audio Playground → Text-to-Speech → Voice Gallery。多种 Neural voices 可选（每个有 voice card：语言支持、性格、风格）。Multilingual voices 可跨多语言使用。输入文本 → 选择语音 → 播放。实际使用通过程序提交文本→获取音频文件→在应用中使用。

## English Short Summary

TTS hands-on: Foundry → Audio Playground → TTS → Voice Gallery. Multiple neural voices available (each with voice card: language support, personality, speaking style). Multilingual voices work across languages. Input text → select voice → play. Actual use: program submits text → gets audio file → uses in application.
