## 中文短总结

Custom Vision Service：当预构建 Vision 服务不满足特定领域需求时，可用自己的图像和标签训练自定义图像模型。两种项目类型：Image Classification（图像分类，给图像贴标签）和 Object Detection（对象检测，返回标签+坐标）。流程：创建项目 → 上传图像 → 标注 → 训练 → 发布 → 调用端点。可通过 Portal 或编程语言使用。

## 考试要点

- Custom Vision = 用自己的数据训练自定义图像模型
- 两种类型：Image Classification / Object Detection

## English Short Summary

Custom Vision Service: build and deploy your own image models using your images and labels when pre-built Vision service doesn't meet domain-specific needs. Two project types: Image Classification (label images) and Object Detection (label + coordinates). Flow: create project → upload images → tag → train → publish → call endpoint. Portal or code.
