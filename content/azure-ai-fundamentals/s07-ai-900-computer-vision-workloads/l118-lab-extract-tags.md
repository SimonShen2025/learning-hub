## 中文短总结

Image Tagging 功能：生成与图像相关的关键词列表（如 indoor, couch, wall, furniture, animal, floor）。vs Captioning（单句描述），Tagging 返回多个关键词。应用：为海量产品图片库生成元数据 → 使图像可搜索和可筛选。50万张产品图无元数据 → 批量通过服务生成 tags → 存入数据库。

## 考试要点

- Image Tagging = 生成图像相关关键词列表
- vs Captioning：关键词列表 vs 单句描述
- 应用：图像库元数据、搜索、筛选
- 批量处理大量图像

## English Short Summary

Image Tagging: produces list of keywords (tags) describing everything relevant in the image (indoor, couch, wall, furniture, animal, floor). Unlike Captioning (single sentence), Tagging returns multiple keywords. Application: generate metadata for massive image libraries → make images searchable and filterable.
