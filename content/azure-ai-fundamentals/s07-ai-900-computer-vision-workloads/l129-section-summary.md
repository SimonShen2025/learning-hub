## 中文短总结

Section 7 总结。**Azure AI Vision**：提取视觉特征（tags, captions, object detection, OCR）。**Face Service**：检测/识别/分析人脸。**Custom Vision**：训练自定义图像模型（Classification / Object Detection），流程：创建项目→上传→标注→训练→发布→调用端点。**Document Intelligence**：从表单/文档提取结构化数据，预构建模型（Invoice, Receipt）。

## 考试要点

- **Vision Service**：tags, captions, object detection, OCR
- **Face Service**：detect, identify, verify, find similar, group
- **Custom Vision**：自定义模型（Classification/Object Detection）
  - 流程：项目→上传→标注→训练→发布→端点
- **Document Intelligence**：结构化文档数据提取
  - 预构建模型：Invoice, Receipt, ID, Business Card

## English Short Summary

Section 7 recap. Azure AI Vision: extract visual features (tags, captions, object detection, OCR). Face Service: detect/recognize/analyze faces. Custom Vision: train custom image models (Classification/Object Detection) — project → upload → tag → train → publish → endpoint. Document Intelligence: extract structured data from forms/documents with pre-built models (Invoice, Receipt).
