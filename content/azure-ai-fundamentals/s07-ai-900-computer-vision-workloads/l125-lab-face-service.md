## 中文短总结

Azure Face Service：检测、识别、分析图像中的人脸。功能：身份验证（identity verification）、人员检测、相似面部查找、面部分组。但大部分高级功能有限制（需向 Microsoft 申请），学生只能使用基础 face detection。在 Foundry → AI Services → Vision → Face → Detect Faces 中测试。返回：face landmarks、面罩检测。考试重点：了解服务能力。

## 考试要点

- Face Service = 检测/识别/分析图像中的人脸
- 功能：detection, identification, verification, find similar, group
- 高级功能有使用限制（需申请）
- 基础功能：face detection + landmarks
- 考试：了解服务能力即可

## English Short Summary

Azure Face Service: detect, recognize, analyze human faces in images. Capabilities: identity verification, people detection, find similar faces, group faces. Most advanced features restricted (requires Microsoft application). Students can only use basic face detection. In Foundry → AI Services → Vision → Face → Detect Faces. Returns: face landmarks, mask detection. Exam: know service capabilities.
