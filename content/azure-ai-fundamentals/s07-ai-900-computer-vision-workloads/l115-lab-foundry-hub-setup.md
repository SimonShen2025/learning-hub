## 中文短总结

创建 Microsoft Foundry Hub 资源（旧版 Azure AI Hub）用于使用 AI 服务。注意：Microsoft 频繁更名（Vision Studio → Azure AI Hub → Microsoft Foundry）。当前使用 Foundry Hub（经典版）因为新版 Foundry 尚不支持所有 Vision 功能的 Playground。创建步骤：搜索 "Azure AI Hub" → 配置（Subscription, RG, Region, Name）→ 部署。

## 考试要点

- Microsoft Foundry Hub = 旧版 Azure AI Hub
- 用于访问和测试 Azure AI Services
- 创建后可使用各种 AI Playground
- 新旧版本并存，功能有差异

## English Short Summary

Create Microsoft Foundry Hub resource (formerly Azure AI Hub) for AI services. Note: Microsoft frequent renames (Vision Studio → Azure AI Hub → Microsoft Foundry). Using Foundry Hub (classic) because new Foundry doesn't support all Vision playgrounds yet. Steps: search "Azure AI Hub" → configure → deploy.
