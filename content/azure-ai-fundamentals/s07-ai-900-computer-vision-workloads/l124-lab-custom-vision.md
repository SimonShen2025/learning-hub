## 中文短总结

Custom Vision 实操。创建资源：搜索 "Custom Vision" → Create options: Both（Training + Prediction）。Training 资源：上传图像/标注/训练模型（需要计算）。Prediction 资源：应用调用模型预测新图像（单独端点和定价）。Free tier 可用于两者。创建后在 Custom Vision Portal 中：创建项目 → 上传图像 → 标注每张图像 → Train → 发布模型。

## 考试要点

- Custom Vision 有两个资源：Training + Prediction
- Training：上传/标注/训练
- Prediction：发布后的模型调用端点
- 两者分离定价
- Custom Vision Portal 进行操作

## English Short Summary

Custom Vision hands-on. Create resource: search "Custom Vision" → Both (Training + Prediction). Training resource: upload/tag/train models (compute). Prediction resource: apps call model for new images (separate endpoint/pricing). Free tier available for both. In Custom Vision Portal: create project → upload images → tag → Train → publish model.
