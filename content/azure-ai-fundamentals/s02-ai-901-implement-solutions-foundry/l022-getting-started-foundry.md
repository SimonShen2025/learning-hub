## 中文短总结

Microsoft Foundry 是用于构建 AI 应用和代理的统一平台。核心优势：将模型部署、代理开发、数据连接、工具管理、监控和安全集于一体。支持多供应商模型（OpenAI GPT、Anthropic Claude、DeepSeek 等），不被锁定在单一供应商。使用 Foundry 需先有 Azure 订阅。被 80,000+ 企业使用，包括 80% 的 Fortune 500。

## 中文长总结

### Microsoft Foundry 定义
- AI 平台，用于构建 AI 应用和代理
- 将十几种工具/服务合并到一个统一平台
- 功能：部署模型、开发代理、管理数据连接、工具、监控、安全

### 核心特性
- **多供应商模型**：GPT (OpenAI)、Claude (Anthropic)、DeepSeek 等
- **Model Catalog**：浏览和部署各供应商模型
- **Agent 开发**：构建智能代理
- **工具集成**：各种 AI 工具和服务

### 使用前提
- 需要 Azure 订阅（Azure 是微软的云计算平台）
- 需创建 Microsoft Foundry 资源
- 可创建 Azure 免费账号来试用

### 规模
- 80,000+ 企业使用
- 80% Fortune 500 公司采用

## 考试要点

## English Short Summary

Microsoft Foundry is a unified AI platform for building AI-powered applications and agents. It brings model deployment, agent development, data connections, tools, monitoring, and security under one umbrella. Supports multi-vendor models (OpenAI GPT, Anthropic Claude, DeepSeek). Requires an Azure subscription. Used by 80,000+ enterprises including 80% of Fortune 500.
