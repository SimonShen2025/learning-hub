## 中文短总结

Microsoft Foundry Portal (ai.azure.com) 界面概览。有新旧界面切换按钮。由于 AI 发展极快，界面频繁更新（AI Foundry → Microsoft Foundry → 新界面），重要的是理解基础概念而非 UI 细节。核心功能区：Discover（发现模型）、Build（构建）、Operate（运维）。后续将使用 Python 程序调用 Foundry 中的模型。

## 中文长总结

### Foundry Portal 访问
- 创建 Foundry 资源后，进入资源概览页
- 点击 Foundry Portal 链接 → ai.azure.com
- 这是主要工作界面

### 界面演进历史
- AI Foundry → Microsoft Foundry → 新界面（持续更新中）
- AI 领域工具和界面变化极快
- 教学重点：基础概念不变，UI 可能有偏差

### 核心功能区
- **Discover**：发现和浏览可用模型
- **Build**：构建应用和代理
- **Operate**：运维和监控

### 学习建议
- 掌握基础概念比记住 UI 布局重要
- Python 程序调用模型是考试要点
- 即使 UI 变化，API 调用方式基本稳定

## English Short Summary

Microsoft Foundry Portal at ai.azure.com has a rapidly evolving interface (AI Foundry → Microsoft Foundry → new UI). Key areas: Discover (browse models), Build (create apps/agents), Operate (monitor). Focus on base concepts rather than UI specifics since AI tools change rapidly. Python-based model interaction is the stable exam-relevant skill.
