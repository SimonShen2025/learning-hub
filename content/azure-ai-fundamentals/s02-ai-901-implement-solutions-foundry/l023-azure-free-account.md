## 中文短总结

创建 Azure 免费账号的流程：需要邮箱（Gmail 或 GitHub）、电话和信用卡。新用户获得 $200 免费额度（30天有效）。步骤：访问 Azure 免费试用页面 → 创建 Microsoft 账号 → 邮箱验证 → 填写个人信息 → 人机验证 → 填写信用卡信息（用于验证身份，不会自动扣费）→ 完成注册。

## 中文长总结

### 注册前准备
- 邮箱账号（Gmail 或 GitHub 均可）
- 手机（接收验证码）
- 信用卡（身份验证用，不会自动收费）

### 注册步骤
1. 访问 Azure 免费试用页面
2. 输入邮箱 → 如无 Microsoft 账号则创建新的
3. 邮箱验证码验证
4. 填写个人信息和姓名
5. 人机验证（Press & Hold）
6. 填写信用卡信息
7. 勾选同意协议 → 完成注册

### 免费账号权益
- $200 免费额度，30天有效
- 部分服务永久免费（有配额限制）
- 升级为 Pay-As-You-Go 后仍可使用免费层服务

### 注意事项
- 仅限新 Azure 客户
- 信用卡仅用于验证，不会自动升级收费
- 课程 demo 产生的费用通常很少

## English Short Summary

Creating an Azure free account requires an email (Gmail/GitHub), phone, and credit card. New users get $200 free credit for 30 days. Steps: sign up → create Microsoft account → email verification → personal info → identity verification → credit card (for verification only, no auto-charge) → complete registration.
