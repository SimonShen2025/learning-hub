## 中文短总结

Azure Portal (portal.azure.com) 导览。核心概念：Subscription（订阅，计费账户，资源费用计入此处）、All Resources（查看所有已创建资源）、Portal Menu（左侧菜单，常用操作和服务入口）。通过搜索栏可找到任何服务或资源。

## 中文长总结

### Azure Portal 基础
- URL: portal.azure.com
- 使用 Microsoft 账号登录
- 左侧 Portal Menu：常用操作和服务列表

### 关键概念
- **Subscription**：计费账户，所有资源费用计入
- **All Resources**：查看账户中所有已创建的资源
- **搜索栏**：搜索资源、服务等
- **Notifications**：通知（如剩余额度提醒）

## English Short Summary

Azure Portal (portal.azure.com) tour. Key concepts: Subscription (billing account for resource charges), All Resources (view all created resources), Portal Menu (left sidebar with common actions/services), and search bar for finding any service or resource.
