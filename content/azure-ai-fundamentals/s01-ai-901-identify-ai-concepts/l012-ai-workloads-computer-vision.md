## 中文短总结

四大 AI 工作负载类型之三：计算机视觉。AI 系统能"看"并从图像/视频中提取含义。核心能力：Image Classification（图像分类：猫/狗、X光正常/异常）、Object Detection（目标检测：定位图像中的特定对象）、Facial Recognition（人脸识别与身份验证）、Document/Image Analysis。应用：制造业质检、零售库存监测、自动驾驶、安防系统。核心主题：将视觉信息转化为结构化可操作数据。

## 中文长总结

### 计算机视觉核心能力

| 能力 | 描述 | 应用场景 |
|------|------|----------|
| Image Classification | 识别图像内容（是什么？） | X光诊断、卫星图像分析 |
| Object Detection | 定位图像中特定对象的位置 | 制造业质检、零售库存、自动驾驶 |
| Facial Recognition | 基于面部特征识别和验证个人 | 安防系统、金融身份验证 |
| Document/Image Analysis | 分析文档图像提取信息 | OCR、文档处理 |

### 关键应用场景
- **制造业**：生产线摄像头实时检测缺陷产品
- **零售**：监测货架库存水平
- **自动驾驶**：识别行人、交通标志
- **安防**：身份验证和监控

### 核心主题
将视觉信息转化为可理解的、结构化的、可操作的数据

## 考试要点

- Computer Vision = AI 系统理解图像和视频
- Image Classification：识别图像内容（这是什么？）
- Object Detection：定位图像中对象的位置（在哪里？）
- Facial Recognition：基于面部特征识别身份
- 场景识别：图像、视频、摄像头 → Computer Vision 工作负载

## English Short Summary

Computer Vision enables AI to interpret images and video. Core capabilities: Image Classification (what is it?), Object Detection (where is it?), Facial Recognition (who is it?), Document Analysis. Applied in manufacturing QC, retail inventory, autonomous vehicles, and security. Core theme: converting visual information into structured, actionable data.
