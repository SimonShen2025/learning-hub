## 中文短总结

本讲演示了在 AWS 控制台创建 MySQL RDS 数据库的完整流程，包括选择引擎、模板（Free Tier）、配置实例、启用 Storage Auto Scaling、设置公共访问和安全组。演示了使用 SQLectron 客户端连接数据库并执行 SQL 操作（CREATE TABLE / INSERT / SELECT）。还展示了创建 Read Replica、查看监控指标（CPU 利用率、连接数）、创建 Snapshot 和迁移 Snapshot 等功能。最后演示了删除数据库的流程（先关闭 Deletion Protection）。

## English Short Summary

This hands-on lecture demonstrates creating a MySQL RDS database in the AWS console, covering engine selection, Free Tier template, instance configuration, Storage Auto Scaling, public access, and security groups. It shows connecting via SQLectron to run SQL operations, exploring Read Replica creation, monitoring metrics (CPU utilization, connection counts), snapshots, and database deletion (disabling deletion protection first).
