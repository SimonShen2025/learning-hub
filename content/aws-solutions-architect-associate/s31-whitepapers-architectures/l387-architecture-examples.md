## 中文短总结

AWS 架构参考资源：①**AWS Architecture Center**（aws.amazon.com/architecture）— 2000+ 架构图和最佳实践文档②**AWS Solutions Library**（aws.amazon.com/solutions）— 预制架构方案，多数提供 **CloudFormation 模板**直接部署，如 Instance Scheduler、AWS Landing Zone。这些是学习架构设计的宝贵资源，考试中有助于理解典型的 AWS 架构模式。

## 中文长总结

两个重要的架构学习资源：

**1. AWS Architecture Center**
- 地址：aws.amazon.com/architecture
- 内容：2000+ 架构参考图
- 涵盖：各行业/各服务的架构最佳实践
- 用途：学习和参考

**2. AWS Solutions Library**
- 地址：aws.amazon.com/solutions
- 内容：预制解决方案
- 特点：**大多提供 CloudFormation 模板一键部署**
- 例子：
  - Instance Scheduler（定时启停 EC2/RDS）
  - AWS Landing Zone（多账号环境配置）

## 考试要点

- Architecture Center = 2000+ 架构图参考
- Solutions Library = 预制方案 + CloudFormation 模板
- 了解这些资源存在即可

## English Short Summary

AWS architecture reference resources: (1) **AWS Architecture Center** (aws.amazon.com/architecture) — 2000+ architecture diagrams and best practices; (2) **AWS Solutions Library** (aws.amazon.com/solutions) — pre-built architectural solutions, most with **CloudFormation templates** for direct deployment (e.g., Instance Scheduler, AWS Landing Zone). Valuable resources for learning AWS architecture patterns.
