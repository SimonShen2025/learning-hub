## 中文短总结

创建 AWS 账户的步骤：填写 root 用户邮箱和账户名、设置密码、选择 Free 或 Paid 计划。Free 计划提供约 $100 credit，无需实际付费，但仍需提供信用卡用于验证（仅扣 $1 验证后退还）。登录时使用 root 用户邮箱。

## 中文长总结

### 账户创建流程
1. 在 AWS 官网点击 Create Account
2. 填写 root 用户邮箱和账户名称
3. 验证邮箱，设置密码（至少 8 字符，需含大写、小写、数字、特殊字符中的三种）
4. 选择账户计划：
   - **Free Plan**：不需绑定信用卡用于计费，提供 $100 credit（最多 $200），credit 用完账户自动关闭
   - **Paid Plan**：用于生产环境，绑定信用卡，credit 用完后从信用卡扣费
5. 填写联系信息、提供信用卡（用于身份验证，仅扣 $1 后退还）
6. 通过手机号验证身份
7. 选择 Basic Support（免费），完成注册

### 登录方式
- 使用 root 用户邮箱登录 AWS Console
- 首次登录可能看到安全提示，课程后续会处理

## English Short Summary

Create an AWS account: provide root email, set password (8+ chars, 3 of 4 character types), choose Free plan (~$100 credit, no charges), provide credit card for verification only ($1 refunded). Confirm identity via phone, select Basic Support (free). Sign in using root user email.
