## 中文长总结

### 课程概述
- 目标：通过 AWS Certified Solutions Architect Associate (SAA-C03) 认证考试
- 时长：约 25 小时，覆盖 30+ AWS 服务
- 适合初学者，可多次重复学习

### AWS 简介
- AWS = Amazon Web Services，全球领先的云服务提供商
- 核心优势：按需获取（on-demand）、弹性扩展（scale easily）
- 与传统本地数据中心的区别：无需物理采购和安装服务器
- 客户案例：Amazon.com、Netflix（不拥有任何服务器，全部使用 AWS）

### 讲师背景
- Stéphane Maarek，SAA-C03 考试得分 96.1%
- 持有 9 项 AWS 认证，曾任 IT 顾问、Solutions Architect、Developer、SysOps

## English Short Summary
