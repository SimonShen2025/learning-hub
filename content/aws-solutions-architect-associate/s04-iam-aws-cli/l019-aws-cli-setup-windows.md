## 中文短总结

在 Windows 上安装 AWS CLI v2：搜索 "aws CLI install windows"，下载 MSI 安装程序，运行安装向导完成安装。安装后在命令提示符中运行 `aws --version` 验证，输出以 "aws-cli/2." 开头即表示成功。升级时重新下载并运行 MSI 即可。

## English Short Summary

Install AWS CLI v2 on Windows: download the MSI installer from the AWS documentation page, run the installer wizard, then verify with `aws --version` in Command Prompt. Output starting with "aws-cli/2." confirms successful installation. To upgrade, simply re-download and re-run the MSI installer.
