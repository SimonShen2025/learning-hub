## 中文短总结

AWS 控制台新增多会话支持（Multi-session Support）功能，可在同一浏览器中同时登录不同 AWS 账户或角色。每个会话窗口独立，资源互不影响。无需再使用隐私窗口来实现多账户并行操作。

## English Short Summary

AWS Console now supports multi-session sign-in, allowing simultaneous login to different AWS accounts or roles in the same browser. Each session window operates independently with isolated resources, eliminating the need for private/incognito windows for parallel account access.
