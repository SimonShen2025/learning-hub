## 中文短总结

在 Mac 上安装 AWS CLI v2：下载 pkg 文件并运行图形化安装程序，按提示完成安装。安装后在终端运行 `aws --version` 验证，输出显示 "aws-cli/2." 即表示安装成功。

## English Short Summary

Install AWS CLI v2 on Mac: download the pkg file from AWS documentation, run the graphical installer, and verify with `aws --version` in the terminal. Output showing "aws-cli/2." confirms successful installation.
