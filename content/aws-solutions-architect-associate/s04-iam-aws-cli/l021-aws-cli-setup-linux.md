## 中文短总结

在 Linux 上安装 AWS CLI v2：依次运行三条命令——下载 zip 文件（curl）、解压（unzip）、以 root 权限运行安装脚本（sudo ./install）。安装后运行 `aws --version` 验证，输出以 "aws-cli/2." 开头即表示成功。

## English Short Summary

Install AWS CLI v2 on Linux: run three commands sequentially—download the zip file (curl), unzip it, and run the installer with sudo. Verify with `aws --version`; output starting with "aws-cli/2." confirms successful installation.
