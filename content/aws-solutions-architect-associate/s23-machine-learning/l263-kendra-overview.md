## 中文短总结

Amazon Kendra 是 ML 驱动的完全托管文档搜索服务，从文档中提取答案。支持多种数据源（文本/PDF/HTML/PowerPoint/Word/FAQ 等）→ 构建 ML 知识索引。提供自然语言搜索能力（如提问"IT 支持台在哪里？"→ 回答"1 楼"）。通过用户交互反馈增量学习（Incremental Learning），可按数据重要性/新鲜度/自定义过滤器调优结果。**考试关键词：文档搜索服务 → Kendra**。

## 中文长总结

Amazon Kendra 是一项完全托管的文档搜索服务，由机器学习驱动。

**核心功能：**
- 从文档中提取答案
- 构建内部 ML 知识索引（Knowledge Index）
- 提供**自然语言搜索**能力（类似 Google 搜索体验）

**支持的数据源：**
- 文本文件、PDF、HTML
- PowerPoint、Microsoft Word
- FAQ 文档
- 其他多种格式

**智能特性：**
- **自然语言问答**：用户提问"IT 支持台在哪里？"→ Kendra 回答"1 楼"
  - 基于索引的所有文档资源提取答案
- **增量学习（Incremental Learning）**：从用户交互和反馈中学习，提升优选搜索结果
- **结果调优**：按数据重要性、新鲜度、自定义过滤器微调搜索结果

## 考试要点

- **文档搜索服务 → Kendra**（考试核心映射）
- ML 驱动的知识索引
- 自然语言搜索，可从文档中提取答案
- 增量学习提升搜索质量

## English Short Summary

Amazon Kendra is a fully managed ML-powered document search service that extracts answers from documents (text/PDF/HTML/PowerPoint/Word/FAQs), building an internal knowledge index. Provides natural language search (e.g., "Where is the IT support desk?" → "1st floor"). Supports incremental learning from user interactions to improve results, with tuning by data importance, freshness, and custom filters. **Exam key: document search service → Kendra.**
