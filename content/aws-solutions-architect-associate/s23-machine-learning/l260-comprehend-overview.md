## 中文短总结

Amazon Comprehend 是完全托管的无服务器 NLP（自然语言处理）服务。功能：语言识别、关键短语/人物/品牌/事件提取、情感分析（正面/负面）、文本分词与词性分析、按主题自动分组文档。用例：分析客户互动邮件了解正负面体验、按主题自动归类文章。**考试关键词：看到 NLP → 选 Comprehend**。

## 中文长总结

Amazon Comprehend 是一项完全托管的无服务器自然语言处理（NLP）服务，使用机器学习从文本中发现洞察和关系。

**核心功能：**
- **语言识别**：检测文本所用语言
- **关键短语提取**：识别地点、人物、品牌、事件
- **情感分析（Sentiment Analysis）**：判断文本正面或负面倾向
- **文本分析**：分词（Tokenization）和词性标注
- **主题建模**：自动发现并按主题归类文档集合

**核心价值：** 将非结构化文本数据转化为结构化洞察

**用例：**
- 分析客户互动：分析支持邮件 → 了解哪些因素导致正面/负面体验 → 改进业务
- 文章自动分类：将大量文章输入 Comprehend → 自动按发现的主题分组

## 考试要点

- **NLP = Comprehend**（考试核心映射）

- 情感分析、关键短语提取、语言识别、主题建模

## English Short Summary

Amazon Comprehend is a fully managed serverless NLP (Natural Language Processing) service using ML to find insights in text. Features: language detection, key phrase/entity extraction (places, people, brands, events), sentiment analysis (positive/negative), tokenization and part-of-speech analysis, and automatic topic modeling for document collections. **Exam key: NLP → Comprehend.**
