## 中文短总结

Amazon Textract 使用 AI/ML 从扫描文档中提取文本、手写内容和数据。支持表单和表格数据提取，可读取 PDF 和图像。用例：金融服务（发票/财务报告处理）、医疗（病历/保险理赔）、公共部门（税表/身份证/护照）。

## 中文长总结

Amazon Textract 是一项使用 AI/ML 的文档数据提取服务。

**核心功能：**
- 从扫描文档中提取：文本、手写内容、结构化数据
- 支持表单（Forms）和表格（Tables）数据提取
- 可读取 PDF 和图像格式

**示例：** 上传驾照 → 自动分析提取出生日期、文件 ID 等字段

**用例：**
- **金融服务**：发票处理、财务报告分析
- **医疗健康**：病历处理、保险理赔
- **公共部门**：税表、身份证件、护照处理

## 考试要点

- Textract = 文档文本和数据提取（OCR + 结构化数据）
- 支持表单、表格、PDF、图像
- 用于发票/身份证/护照/病历等文档处理

## English Short Summary

Amazon Textract uses AI/ML to extract text, handwriting, and data from scanned documents including forms and tables, supporting PDFs and images. Use cases: financial services (invoices/financial reports), healthcare (medical records/insurance claims), and public sector (tax forms/ID documents/passports).
