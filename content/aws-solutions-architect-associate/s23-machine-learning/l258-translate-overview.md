## 中文短总结

Amazon Translate 提供自然准确的语言翻译服务，支持大规模文本高效翻译。用例：网站和应用程序内容本地化，为国际用户提供多语言支持。

## 中文长总结

Amazon Translate 是一项基于机器学习的语言翻译服务。

**核心特性：**
- 提供自然准确的语言翻译
- 支持大量文本的高效翻译
- 多语言支持广泛

**主要用例：**
- 网站和应用程序内容本地化
- 为国际用户提供多语言支持
- 大规模文档翻译

## 考试要点

- Translate = 语言翻译服务

## English Short Summary

Amazon Translate provides natural and accurate language translation for localizing websites and applications for international users, supporting efficient large-volume text translation.
