## 中文短总结

Amazon Personalize 是完全托管 ML 服务，提供实时个性化推荐（与 Amazon.com 使用相同技术）。功能：个性化产品推荐、重新排序、定制化直接营销。输入数据：S3（用户交互等）+ Personalize API（实时数据）。输出：定制个性化 API 供网站/移动应用/SMS/邮件使用。数天即可构建（非数月），无需手动构建/训练/部署 ML 方案。**考试关键词：个性化推荐 → Personalize**。

## 中文长总结

Amazon Personalize 是一项完全托管的 ML 服务，为应用提供实时个性化推荐。

**核心特性：**
- 与 Amazon.com 购物推荐使用**相同技术**
- 数天即可构建推荐模型（非数月）
- 无需手动构建、训练和部署 ML 解决方案

**推荐类型：**
- 个性化产品推荐
- 搜索结果重新排序（Re-ranking）
- 定制化直接营销

**架构：**
- 输入：
  - S3（用户交互历史数据等）
  - Personalize API（实时数据集成）
- 输出：
  - 定制化个性化 API → 网站、移动应用
  - SMS、邮件推送

**用例：** 零售商店、媒体和娱乐行业

## 考试要点

- **个性化推荐 → Personalize**（考试核心映射）
- 与 Amazon.com 相同技术
- 数天即可构建，无需手动 ML
- 输入：S3 + API，输出：个性化推荐 API

## English Short Summary

Amazon Personalize is a fully managed ML service for real-time personalized recommendations (same technology as Amazon.com). Features: personalized product recommendations, re-ranking, and customized direct marketing. Input: S3 (user interactions) + Personalize API (real-time). Output: customized personalized API for websites/mobile apps/SMS/emails. Takes days, not months, with no need to build/train/deploy ML solutions manually. **Exam key: personalized recommendations → Personalize.**
