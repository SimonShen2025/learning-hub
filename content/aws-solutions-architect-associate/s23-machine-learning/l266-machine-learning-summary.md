## 中文短总结

AWS ML 服务速查表：**Rekognition** = 人脸检测/标签/名人识别（图像视频）| **Transcribe** = 语音→文本（字幕）| **Polly** = 文本→语音 | **Translate** = 语言翻译 | **Lex** = 聊天机器人/ASR + **Connect** = 云联络中心 | **Comprehend** = NLP 自然语言处理 | **SageMaker** = 自定义 ML 模型构建平台 | **Kendra** = ML 文档搜索引擎 | **Personalize** = 实时个性化推荐 | **Textract** = 文档文本/数据提取。

## 中文长总结

AWS 机器学习服务完整汇总（考试必背）：

| 服务 | 功能 | 关键词 |
|------|------|--------|
| **Rekognition** | 图像/视频中的人脸检测、标签、名人识别 | 图像分析 |
| **Transcribe** | 语音→文本（ASR），可生成字幕 | 语音转文本 |
| **Polly** | 文本→语音（TTS），Lexicon + SSML | 文本转语音 |
| **Translate** | 语言翻译 | 翻译 |
| **Lex** | ASR + NLU → 聊天机器人 | 聊天机器人 |
| **Connect** | 云端联络中心，比传统便宜 80% | 联络中心 |
| **Comprehend** | NLP 自然语言处理 | NLP |
| **SageMaker** | 构建自定义 ML 模型（开发者/数据科学家） | 自定义 ML |
| **Kendra** | ML 驱动的文档搜索引擎 | 文档搜索 |
| **Personalize** | 实时个性化推荐（同 Amazon.com） | 推荐 |
| **Textract** | 从文档中提取文本和数据 | OCR/文档提取 |

## 考试要点

- 记住每个服务的一句话定义和关键词映射
- Transcribe vs Polly：方向相反（语音↔文本）
- Lex + Connect = 智能联络中心组合
- NLP → Comprehend
- 文档搜索 → Kendra
- 个性化推荐 → Personalize
- 自定义 ML → SageMaker
