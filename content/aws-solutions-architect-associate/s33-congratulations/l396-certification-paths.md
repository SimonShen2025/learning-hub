## 中文短总结

AWS 认证路径总览：根据职业角色推荐不同认证路径。**架构师路径**：Cloud Practitioner → Solutions Architect Associate → Solutions Architect Professional → Security Specialty。**应用架构**：+Developer Associate → DevOps Engineer Professional。**运维**：SysOps Administrator → DevOps Engineer。**安全**：SysOps → Security Specialty → Advanced Networking。**开发**：Developer → DevOps Engineer。**数据**：Solutions Architect Associate → Data Engineer。**AI/ML**：AI Practitioner → ML Engineer Associate → ML Specialty。还包括 DevSecOps、网络工程师、数据科学家、Prompt Engineer 等路径。所有路径详见 AWS 官方认证页面。

## 中文长总结

AWS 认证体系按角色推荐路径：

**认证级别：**
- Foundational：Cloud Practitioner / AI Practitioner
- Associate：Solutions Architect / Developer / SysOps / Data Engineer / ML Engineer
- Professional：Solutions Architect Professional / DevOps Engineer
- Specialty：Security / Advanced Networking / ML

**主要职业路径：**

| 角色 | 推荐路径 |
|------|---------|
| 解决方案架构师 | CP → SAA → SAP → Security |
| 开发者 | CP → Developer → DevOps |
| 运维工程师 | CP → SysOps → DevOps |
| 安全工程师 | CP → SysOps → Security |
| 数据工程师 | CP → SAA → Data Engineer |
| ML 工程师 | CP → AI Practitioner → SAA → ML |
| DevSecOps | CP → SysOps → DevOps → Security |
| 网络工程师 | CP → SAA → Advanced Networking |

## 考试要点

- 了解 AWS 认证体系结构即可
- SAA 是多条路径的基础
- Professional 和 Specialty 是进阶认证

## English Short Summary

AWS certification paths by career role: **Architect**: Cloud Practitioner → SAA → SA Professional → Security Specialty. **Developer**: → Developer Associate → DevOps Engineer. **Operations**: → SysOps → DevOps. **Security**: → SysOps → Security Specialty. **Data**: → SAA → Data Engineer. **AI/ML**: → AI Practitioner → ML Engineer → ML Specialty. Also covers DevSecOps, network engineer, data scientist, prompt engineer paths. SAA is foundational to many paths.
