## 中文短总结

Amazon SES（Simple Email Service）：全托管电子邮件服务，安全大规模发送邮件。应用通过 SES API 或 SMTP 服务器发送批量邮件。支持出站和入站邮件。提供：①声誉仪表板（打开率）②性能洞察③反垃圾邮件反馈④邮件统计（投递/退回/反馈/打开率）⑤DKIM + SPF 安全标准⑥灵活 IP 部署（共享/专用/自有）。用途：交易邮件、营销邮件、批量通信。

## English Short Summary

Amazon SES (Simple Email Service): fully managed email service for secure, global, at-scale email sending. Uses SES API or SMTP server. Supports outbound and inbound emails. Features: reputation dashboard, performance insights, anti-spam feedback, email statistics (delivery/bounce/open), DKIM + SPF security, flexible IP deployment (shared/dedicated/customer-owned). Use cases: transactional, marketing, and bulk email communications.
