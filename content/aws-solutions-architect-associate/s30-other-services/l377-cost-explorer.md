## 中文短总结

AWS Cost Explorer：可视化、理解和管理 AWS 成本与用量。功能：①自定义报告分析成本②高级别（全账号总成本）到细粒度（月/小时/资源级别）③**Savings Plan 推荐**（基于使用量优化价格计划）④**使用量预测**（基于历史数据预测未来 18 个月）。按 AWS 服务/实例类型/时间维度查看成本分布。可能是考试中唯一的计费服务题。

## 中文长总结

Cost Explorer 是 AWS 主要的成本分析工具。

**功能概览：**

| 功能 | 说明 |
|------|------|
| **自定义报告** | 按服务/账号/时间分析 |
| **粒度** | 全账号 → 月 → 小时 → 资源级 |
| **Savings Plan** | 基于用量推荐最优节省计划 |
| **预测** | 历史数据 → 预测未来 18 个月 |

**典型用法：**
- 月度服务成本图 → 找最贵的服务
- 小时级资源成本 → 找低使用率实例
- Savings Plan 弹窗 → 推荐最优计划

## 考试要点

- Cost Explorer = 成本可视化 + 报告
- Savings Plan 推荐（基于实际使用量）
- 预测未来 18 个月
- 可能是 SAA 考试唯一的计费题来源

## English Short Summary

AWS Cost Explorer: visualize, understand, and manage AWS costs and usage. Features: custom reports, multi-granularity (all accounts → monthly → hourly → resource level), **Savings Plan recommendations** (optimize pricing based on usage), **usage forecasting** (up to 18 months based on history). Likely the only billing service tested on the SAA exam.
