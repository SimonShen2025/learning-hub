## 中文短总结

实操演示 Amazon Kinesis 控制台中的数据分析选项。Streaming Application 基于 Apache Flink（选择 Flink 运行时版本，上传应用代码，可通过 Flink Dashboard 监控）。Studio Notebook 可快速创建笔记本运行流式应用。SQL Applications（Legacy）支持从 Kinesis Data Firehose 读取，但 AWS 目前推荐统一使用 Kinesis Data Analytics Studio + Apache Flink。

## English Short Summary

Hands-on overview of Kinesis analytics options in the console. Streaming Application uses Apache Flink (select runtime version, upload application code, monitor via Flink Dashboard). Studio Notebook creates quick notebooks for streaming applications. SQL Applications (Legacy) supports reading from Kinesis Data Firehose, but AWS now recommends using Kinesis Data Analytics Studio with Apache Flink for all new workloads.
