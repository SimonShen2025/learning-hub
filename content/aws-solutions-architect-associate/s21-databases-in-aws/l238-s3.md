## 中文短总结

从数据库视角看 S3：Key-Value 对象存储，适合大对象（最大 5TB），不适合大量小对象。无服务器，无限扩展。核心功能：版本控制、多种加密（SSE-S3/SSE-KMS/SSE-C/客户端）、复制、MFA Delete、Access Points、Object Lambda、Object Lock/Vault Lock。性能优化：Multi-Part Upload、Transfer Acceleration、S3 Select。自动化：S3 Event Notifications (SNS/SQS/Lambda/EventBridge)。批量操作：S3 Batch + S3 Inventory。用例：静态文件、大文件 Key-Value、网站托管。

## 中文长总结

### S3 作为数据库的定位

| 特性 | 说明 |
|------|------|
| 类型 | **Key-Value 对象存储** |
| 适合 | **大对象**（最大 5TB/对象） |
| 不适合 | 大量小对象 |
| 扩展 | **无服务器、无限扩展** |

### 存储层级
S3 Standard → Infrequent Access → Intelligent → Glacier
- **生命周期策略**自动转换层级

### 安全功能

| 类别 | 功能 |
|------|------|
| 访问控制 | IAM + Bucket Policy + ACL + **Access Points** |
| 加密 | SSE-S3, SSE-KMS, SSE-C, 客户端加密, TLS 传输加密 |
| 数据保护 | 版本控制, MFA Delete, **Object Lock / Vault Lock** |
| 数据处理 | **S3 Object Lambda**（修改对象后再返回应用） |
| 跨域 | CORS |

### 性能优化

| 功能 | 说明 |
|------|------|
| **Multi-Part Upload** | 并行上传大文件 |
| **Transfer Acceleration** | 跨区域加速传输 |
| **S3 Select** | 只检索需要的数据 |

### 批量操作
- **S3 Batch** — 批量操作（加密未加密对象、跨桶复制等）
- **S3 Inventory** — 生成文件列表供 Batch 使用

### 自动化
- **S3 Event Notifications** → SNS / SQS / Lambda / EventBridge

## 考试要点

- S3 = **大对象 Key-Value 存储**（最大 5TB），不适合小对象
- 加密方式：SSE-S3 / SSE-KMS / SSE-C / 客户端加密
- **Object Lock** = 防篡改，**Vault Lock** = Glacier 防篡改
- **S3 Batch** = 批量加密/复制操作
- **Transfer Acceleration** = 跨区域加速
- **S3 Select** = 只检索需要的数据
- **S3 Event Notifications** = 对象创建事件自动化
- 用例：静态文件、大文件存储、网站托管

## English Short Summary

S3 from a database perspective: key-value object store for large objects (max 5TB/object), not ideal for many small objects. Serverless, infinite scaling. Features: versioning, multiple encryption options (SSE-S3/KMS/C/client-side), replication, MFA Delete, Access Points, Object Lambda, Object Lock/Vault Lock. Performance: Multi-Part Upload, Transfer Acceleration, S3 Select. Automation: S3 Event Notifications (SNS/SQS/Lambda/EventBridge). Batch operations: S3 Batch + S3 Inventory. Use cases: static files, large object key-value store, website hosting.
