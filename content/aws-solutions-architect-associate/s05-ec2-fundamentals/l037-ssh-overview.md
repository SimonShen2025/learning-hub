## 中文短总结

SSH 连接 EC2 实例的方式概述：Mac/Linux/Win10+ 可直接使用 SSH 命令行；Win7/8 使用 PuTTY（功能等同 SSH）；所有平台均可使用 EC2 Instance Connect（浏览器内连接，仅支持 Amazon Linux 2）。SSH 是本课程最容易出问题的环节——遇到问题可参考后续故障排查指南。只要有一种方式能连接即可，不需全部方式都工作。

## English Short Summary

SSH connection methods for EC2: Mac/Linux/Win10+ use the SSH command, Win7/8 use PuTTY (equivalent functionality), and all platforms can use EC2 Instance Connect (browser-based, Amazon Linux 2 only). SSH is the most common stumbling point in the course—troubleshooting guide provided. Only one working method is needed to proceed.
