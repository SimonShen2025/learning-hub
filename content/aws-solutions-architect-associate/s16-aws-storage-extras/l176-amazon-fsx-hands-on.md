## 中文短总结

本讲演示 Amazon FSx 创建流程。可选择四种文件系统：Lustre（HPC Linux，Persistent/Scratch 模式，配置 throughput）、Windows File Server（SMB 协议，Multi-AZ/Single-AZ，SSD/HDD，Active Directory 集成）、NetApp ONTAP（Linux/Windows/macOS 兼容，Multi-AZ/Single-AZ，去重/压缩/紧凑功能）、OpenZFS（ZFS 文件系统）。所有类型都部署在 VPC 中并支持加密。考试只需高层理解各类型差异。

## English Short Summary

Hands-on FSx creation walkthrough. Four file system options: Lustre (HPC Linux, Persistent/Scratch, configurable throughput), Windows File Server (SMB, Multi-AZ/Single-AZ, SSD/HDD, Active Directory integration), NetApp ONTAP (all OS compatible, Multi-AZ/Single-AZ, deduplication/compression/compaction), OpenZFS. All deploy within VPC with encryption. Exam requires high-level understanding of differences between types.
