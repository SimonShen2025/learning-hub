## 中文短总结

考试注册流程实操：①访问 aws.training/certification → 创建 AWS Builder ID②进入认证门户 → Exam Registration → Schedule③选择考试方式：**线下（Test Center）**需要带照片 ID → 或 **线上（OnVUE）**需要电脑+摄像头+网络 → 务必提前运行系统测试 → 桌面必须清空④选择语言和时区 → 选日期时间 → 支付（$150）⑤考试日在 Dashboard → Manage Pearson VUE Account → Launch Exam 开始考试。需提前 30 分钟 Check-in。

## English Short Summary

Exam signup walkthrough: (1) Go to aws.training/certification → create AWS Builder ID; (2) Certification portal → Exam Registration → Schedule; (3) Choose: **in-person (Test Center)** with photo ID, or **online (OnVUE)** with computer + webcam + internet — **run system test early** — desk must be completely clear; (4) Select language, timezone, date/time → pay ($150); (5) On exam day: Dashboard → Manage Pearson VUE Account → Launch Exam. Check in 30 minutes before appointment.
