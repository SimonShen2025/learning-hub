## 中文短总结

考试流程详解：①线上注册 → 支付 $150②考试日提供身份证件（ID/护照）③不能做笔记/用笔/说话④在家考试需扫描房间确保空桌⑤**65 题 / 130 分钟**⑥不确定的题目用 **Flag** 标记，最后可回审⑦提交后不能修改⑧**720/1000 通过**⑨5 天内出结果（通常更快）⑩会收到邮件通知总分（可在认证网站下载）⑪永远不告诉具体哪题对错⑫**挂科 14 天后可重考**。

## 中文长总结

考试流程关键信息一览：

| 项目 | 详情 |
|------|------|
| 费用 | $150 |
| 身份证明 | ID 或护照 |
| 禁止事项 | 笔记/笔/说话 |
| 题数/时间 | **65 题 / 130 分钟** |
| 通过分 | **720 / 1000** |
| 出分时间 | ≤ 5 天（通常更快） |
| 对错反馈 | ❌ 永不告知 |
| 重考等待 | **14 天** |
| Flag 功能 | 标记不确定题目，最后复查 |
| 提交后 | 不可修改 |

## 考试要点

- 65 题 130 分钟 ≈ 2 分钟/题
- 720/1000 通过
- 提交后不能改答案
- 5 天内出结果
- 不告诉哪题对错（防作弊）
- 失败后 14 天可重考

## English Short Summary

Exam logistics: register online → pay $150 → provide photo ID on exam day → no notes/pen/speaking allowed → home exam requires room scan. **65 questions in 130 minutes**. Use **Flag** to mark uncertain questions for review. **Cannot change answers after submission**. **720/1000 to pass**. Results within 5 days (usually faster). Email notification with score (downloadable from cert website). **Never reveals which answers were correct/wrong**. If failed, **retake after 14 days**.
