## 中文短总结

考试省钱技巧：通过任何一门 AWS 认证考试后，会获得 **50% 折扣 Token**，可用于下一次考试。操作：认证门户 → Claim Benefits → 获得 Voucher Code → 下次考试结账时输入 Promo Code → 自动减半费用。Token 有过期日期，需及时使用。

## English Short Summary

Cost-saving trick: after passing any AWS certification exam, you receive a **50% discount token** for your next exam. Steps: Certification portal → Claim Benefits → get Voucher Code → enter Promo Code at checkout → 50% discount applied. Tokens have expiry dates — use them in time.
