## 中文短总结

VPC（Virtual Private Cloud）是 AWS 网络的骨干。本节将从零开始逐步学习 VPC 的所有概念并动手构建完整 VPC。注意：本节会创建一些收费资源（如 NAT Gateway），建议一次性完成所有实操，完成后立即删除资源以节省费用。

## 中文长总结

VPC 是整个 AWS 课程的网络基础设施核心。本节将系统学习：
- CIDR 与 IP 地址
- 子网（公有/私有）
- Internet Gateway 与路由表
- NAT Gateway/NAT Instance
- NACL 与安全组
- VPC Peering、VPC Endpoints
- VPN、Direct Connect
- Transit Gateway 等

建议集中在 1-2 天内完成所有实操，然后删除所有资源以避免不必要的费用。

## 考试要点

- VPC 是 AWS 网络核心，考试重点章节
- 理解完整的 VPC 架构图是关键

## English Short Summary

VPC (Virtual Private Cloud) is the backbone of AWS networking. This section covers building a VPC from the ground up step by step. Note: some resources (NAT Gateway, etc.) incur costs — complete all hands-on in one or two sessions and delete resources afterward.
