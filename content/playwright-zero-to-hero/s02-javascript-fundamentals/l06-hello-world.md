## 中文短总结

创建 Node.js Hello World 应用。`npm init` 初始化项目生成 `package.json`，创建 `.js` 文件，用 `console.log("Hello World")` 输出，通过 `node filename.js` 在终端运行。

## 中文长总结

- 创建项目文件夹，在 VS Code 中打开
- `npm init` 生成 `package.json`（Node.js 项目配置文件），`dependencies` 和 `devDependencies` 用于管理第三方库
- 创建 `.js` 文件，使用 `console.log()` 输出内容
- 终端执行 `node <filename>.js` 运行脚本

## English Short Summary

Create a Node.js Hello World app: `npm init` to generate `package.json`, write `console.log("Hello World")` in a `.js` file, run with `node filename.js`.
