## 中文长总结

### 课程结构

课程按以下顺序组织，每节课建立在前一节的基础上：

1. **Playwright Hands-On Overview** — 安装、命令行运行、UI 模式、文件结构
2. **Web Elements 交互** — 定位元素、assertions、超时配置、值提取
3. **UI 组件** — input fields、radio buttons、checkboxes、date pickers 等
4. **Page Object Model** — 三层架构设计，适用于实际项目
5. **API 测试** — mock APIs、自定义 API 请求、API 预置测试数据、API 认证
6. **高级功能** — fixtures、global setup/teardown、Docker 运行测试

### 学习建议

- 按上传顺序逐课学习，不要跳跃
- 跟着讲师一起写代码，必要时暂停视频
- 有问题查看 Q&A 区或发私信
