## 中文短总结

Desktop App 支持跨项目多 session 并行切换。任务完成后可展开查看编辑了哪些文件及具体 diff。右上角可切换到 Plan 视图查看完整 plan——比 CLI 中的 plan 输出更可视化。

## 中文长总结

### 多 Session 管理

- 同时运行多个 Claude Code session
- 跨不同项目切换
- 侧边栏浏览历史 session

### 任务结果审查

- 展开 output 区域查看修改的文件列表
- 逐文件查看 diff（类似 VS Code integration）
- 比 CLI 纯文本 output 更直观

### Plan Mode 可视化

- 任务 kick-off 时可选 plan mode
- 右上角切换到 **Plan** tab 查看完整 plan
- 审查 plan 后再批准执行

### 适用人群

不喜欢 terminal 的开发者可用 Desktop 获得完整 Claude Code 体验。

## English Short Summary

Desktop App: multi-session across projects, expandable file diffs after tasks, dedicated Plan tab for visual plan review. More intuitive than CLI text output for reviewing AI changes and managing parallel sessions.
