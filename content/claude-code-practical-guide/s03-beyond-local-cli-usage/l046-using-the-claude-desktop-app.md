## 中文短总结

深入 Claude Desktop App 的 Claude Code 模式：安装后可选项目文件夹，在 GUI 中 dispatch 编码任务。支持多 session、可视化 diff。CLI 仍是主力，Desktop 是便捷替代或补充，可与 CLI 同时使用。

## 中文长总结

### Desktop App Claude Code 模式

- 安装 macOS/Windows 版 Claude Desktop
- 切换到 Code 模式
- New Session → 选择 working directory（已有或新项目）
- 在 GUI 输入 prompt dispatch 任务

### 与 CLI 的关系

- 课程主体用 CLI，Desktop 是 GUI 等价物
- 可同时使用 CLI + Desktop
- 功能覆盖大部分 CLI 能力

### 本 Section 重点

Desktop App、Mobile Dispatch、Remote Control、Telegram Channels、Scheduled Tasks 等超越本地 CLI 的场景。

## English Short Summary

Deep dive into Claude Desktop App's Code mode: select project folder, dispatch tasks via GUI. Supports multiple sessions and visual diffs. Complements or replaces CLI—both can run simultaneously. Section 3 covers all beyond-local-CLI features.
