### 两种工作哲学

- **Ralph Loop**：极端自动化，AI 自验证迭代
- **Plan Mode + Human in the loop**：逐步审查，保持控制

两种都 valid——选适合项目和风险偏好的方式。

### 核心 takeaway

软件越来越多由 AI 编写，但**软件工程师**负责 steering：决定写什么、怎么写、审查什么。Claude Code 是 powerful tool，本课程提供了 efficiently 使用它的 complete toolkit。
