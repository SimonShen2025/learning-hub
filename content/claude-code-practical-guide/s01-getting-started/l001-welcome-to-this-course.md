## 中文短总结

Maximilian Schwarzmüller 介绍本课程目标：高效掌握 Claude Code 的基础与高级功能。课程通过完整示例项目演示，而非纯理论讲解，帮助你在真实开发场景中理解 Claude Code 的能力。

## 中文长总结

### 课程定位

Claude Code 是 Anthropic 推出的 AI 编程 CLI 工具。本课程面向希望从「会用」进阶到「用得好」的开发者。

### 教学方式

- 以完整示例项目贯穿全课，展示 Claude Code 各项功能在实际开发中的用法
- 覆盖基础操作与高级特性（subagents、skills、MCP、hooks、plugins 等）
- 强调 agentic engineering：用 AI 生成大量代码，但开发者保持对规则、库选型和代码质量的控制

## English Short Summary

Course intro by Maximilian Schwarzmüller: learn efficient Claude Code usage through a hands-on example project covering basics and advanced features, not just theory. Focus on agentic engineering—AI-assisted coding with developer oversight.
