## 中文短总结

Claude Code 是 CLI 驱动的 AI 编程工具。课程区分 vibe coding 与 agentic engineering，并规划从安装配置、上下文管理、subagents/skills，到 MCP、hooks、plugins 等高级特性的完整学习路径。

## 中文长总结

### 两种使用哲学

- **Vibe coding**：极度依赖 AI，开发者介入较少
- **Agentic engineering（推荐）**：AI 生成大部分代码，但开发者明确指定规则、库、模式，并审查输出

### 课程结构

1. **Getting Started**：安装、配置、基础用法、权限、沙箱
2. **Key Features**：prompt/context engineering、Plan Mode、subagents、skills、MCP、hooks、plugins
3. **Beyond Local CLI**：Desktop App、Remote Control、移动端 Dispatch、定时任务等

### 核心目标

提供全面工具集，提高用 Claude Code 产出高质量代码的概率，并分享讲师实际工作中的使用模式与适用场景。

## English Short Summary

Claude Code is a CLI AI coding tool. Course contrasts vibe coding vs agentic engineering (staying in control). Covers setup, context management, subagents, skills, MCP, hooks, plugins, and real-world usage patterns via a demo project.
