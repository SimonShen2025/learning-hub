## 中文短总结

Claude Desktop（macOS/Windows）除普通 Chat 和 Cowork 模式外，也包含 Claude Code 模式。Mobile App 同样支持 Code 模式，可 dispatch 任务到云端或本地。课程主体聚焦 CLI，后续章节再深入 Desktop/Mobile 功能。

## 中文长总结

### Claude Desktop App

- 多种模式：普通 Chat（类 ChatGPT）、Cowork（通用工作助手）、**Claude Code**
- 可选项目文件夹作为 working directory，在 GUI 中 dispatch 编码任务
- 可作为 CLI 的替代跟练方式

### Claude Mobile App

- 同样包含 Chat 和 Code 模式
- 支持 dispatch/manage Cloud 任务，以及 dispatch 到本地系统
- 后续课程会演示 Remote Control 等跨设备场景

### 定位

CLI 是 Claude Code 主产品、大多数开发者日常使用的方式；Desktop/Mobile 是扩展入口，Section 3 会详细覆盖。

## English Short Summary

Claude Desktop (macOS/Windows) includes Chat, Cowork, and Claude Code modes—select a folder and dispatch coding tasks. Mobile app also has Code mode for cloud/local task dispatch. Course focuses on CLI first; Desktop/Mobile covered later in Section 3.
