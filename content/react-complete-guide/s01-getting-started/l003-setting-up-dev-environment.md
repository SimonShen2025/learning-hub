## Summary

Step-by-step guide to setting up a React project using Vite — the recommended modern toolchain for new React apps.

## Key Concepts

- Node.js and npm are prerequisites
- Vite replaces the older Create React App (CRA) toolchain
- The development server supports Hot Module Replacement (HMR)

## Detailed Notes

**Prerequisites:**
- Install [Node.js](https://nodejs.org/) (LTS version recommended)
- Verify: `node -v` and `npm -v`

**Create a new project:**

```bash
npm create vite@latest my-react-app -- --template react
cd my-react-app
npm install
npm run dev
```

The project structure generated by Vite:
- `src/` — your application source code
- `src/main.jsx` — entry point
- `src/App.jsx` — root component
- `public/` — static assets

## Code Examples

```jsx
// src/main.jsx
import { StrictMode } from 'react'
import { createRoot } from 'react-dom/client'
import App from './App.jsx'

createRoot(document.getElementById('root')).render(
  <StrictMode>
    <App />
  </StrictMode>,
)
```
