## 中文短总结

CRA 创建 usePopcorn 项目。查看最终版本：搜索电影 → 显示搜索结果 → 点击查看详情 → 添加评分 → 管理已看列表 → 统计平均分/时长。导入 starter CSS 和临时数据。

## 中文长总结

### 项目功能预览
- 顶部：搜索框 + 结果计数
- 左面板：搜索结果列表（电影海报、标题、年份）
- 右面板：电影详情 / 已观看列表
- 评分系统（Star Rating 组件）
- 统计：平均 IMDb 评分、用户评分、电影时长

### 初始设置
- CRA 创建项目
- 导入 CSS
- 临时静态电影数据（后续替换为 API 数据）
- 基础组件结构搭建
