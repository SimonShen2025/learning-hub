## 中文短总结

React 官方文档（react.dev）结构：**Learn** 部分包含基础教程和进阶内容（特别推荐后续学 effects 时阅读 "Escape Hatches" 部分）；**Reference** 部分是 API 全面参考，每个 hook/方法都有详细说明和示例（如 `useState`、`useLayoutEffect`）。建议学习时常开文档页面随时查阅。

## 中文长总结

### react.dev 文档结构
- **Learn**：入门教程和 React 基础知识。重点推荐 **Escape Hatches** 部分——在学到 effects 进阶内容时阅读，质量很高。
- **Reference**：每个 React API 的完整说明。如 `useState` 页面包含用法说明、参数、返回值和各种示例。

### 使用建议
- 学习期间保持文档页面常开。
- 课程中未深入的 API（如 `useLayoutEffect`）可通过文档自行学习。
- 好的开发者知道在哪里找到所需信息，官方文档通常是最佳来源。

## English Short Summary

React docs at react.dev: **Learn** section for tutorials (especially "Escape Hatches" for advanced effects), **Reference** section for complete API docs on every hook/method. Keep docs open while learning — good developers know where to find information.
