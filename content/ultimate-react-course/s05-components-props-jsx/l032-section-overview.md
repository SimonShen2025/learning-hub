## 中文短总结

本 section 介绍 React 三个核心概念：组件（Components）、属性（Props）和 JSX。通过构建 Pizza Menu 项目，学习如何创建和复用组件、使用 JSX 语法、通过 Props 在组件间传递数据、渲染列表和条件渲染。

## 中文长总结

Section 涵盖内容：
- 组件是 React 应用的构建块（building blocks）
- 使用 JSX 描述组件的外观
- 通过 Props 从父组件向子组件传递数据
- 渲染列表（rendering lists）
- 条件渲染（conditional rendering）
- 实战项目：Pizza Menu

## English Short Summary

Covers three core React concepts via the Pizza Menu project: Components (building blocks of UI), Props (data passing from parent to child), and JSX (syntax for describing component appearance). Also covers rendering lists and conditional rendering.
