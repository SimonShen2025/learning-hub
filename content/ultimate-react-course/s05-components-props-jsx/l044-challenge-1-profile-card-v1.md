## 中文短总结

编程挑战：构建开发者 Profile Card。练习组件创建、props 传递、JSX 嵌入表达式。创建 `Avatar`、`Intro`、`SkillList` 组件，通过 props 传递头像 URL、姓名、简介等数据。第一版使用基础 props，第二版（Challenge #2）将使用数组和列表渲染。

## 中文长总结

### 挑战内容
- 创建开发者名片组件
- 包含头像（Avatar）、介绍（Intro）、技能列表（SkillList）
- 使用 props 传递数据（而非硬编码）

### 练习要点
- 组件拆分与组合
- Props 传递不同类型的数据（字符串、图片路径）
- JSX 中使用 `{}` 嵌入 JavaScript 表达式
- 应用之前学到的所有概念

## English Short Summary

Coding challenge: build a developer Profile Card with Avatar, Intro, and SkillList components. Practice component creation, props passing, and JSX expressions. Version 1 uses basic props; version 2 (Challenge #2) adds array rendering.
