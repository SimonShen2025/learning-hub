## 中文短总结

开始构建"Far Away"旅行打包清单项目 — 一个有真正交互功能的 React 应用。可以添加物品、标记已打包、删除物品、排序和统计。CRA 创建新项目，从提供的 CSS 文件开始。这是本 section 剩余课程的实战项目。

## 中文长总结

### 项目功能
- 添加打包物品（名称 + 数量）
- 标记物品为"已打包"
- 删除物品
- 显示打包进度（已打包 X / 总共 Y）
- 列表排序（按输入顺序/名称/打包状态）
- 清空列表

### 项目目标
- 实践 state 管理（多个 state 协同工作）
- 学习表单处理和受控组件
- 理解 state 提升（lifting state up，后续 section）
- 练习列表渲染 + 条件渲染

## English Short Summary

Start the "Far Away" travel packing list project — a fully interactive React app with add items, mark packed, delete, sort, and progress stats. Created with CRA. Practice project for state management, form handling, and controlled elements throughout the remaining lectures.
