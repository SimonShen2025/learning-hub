## 中文短总结

React Router v6.4+ 数据加载功能。新项目 Fast React Pizza Co.。学习 loaders（读取数据）、actions（写入数据）、fetcher（无导航的数据获取/更新）。现代 React Router 不仅是路由 — 还是数据层。

## 中文长总结

### 核心功能
1. **Loaders** — 路由加载前获取数据
2. **Actions** — 表单提交处理（写入/更新数据）
3. **Error elements** — 路由级错误处理
4. **Fetcher** — 无导航的数据获取和更新
5. **Loading indicators** — 加载状态显示

### 项目：Fast React Pizza Co.
- 在线比萨订餐应用
- 查看菜单、创建订单、追踪订单
- 购物车使用 Redux
- 数据获取使用 React Router
