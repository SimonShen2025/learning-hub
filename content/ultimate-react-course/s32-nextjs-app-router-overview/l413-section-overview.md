Next.js App Router 综述。从 SSR 概念开始，手动实现 SSR + Hydration，然后学习 Next.js 框架本身：项目设置、路由、布局、React Server Components (RSC)、数据获取、客户端组件、加载状态。

### 核心概念预览
- **SSR** — 服务端渲染 HTML → 客户端 hydrate
- **RSC** — 组件在服务端运行，不发送 JS 到客户端
- **App Router** — 基于文件系统的路由（`app/` 目录）
- **Server vs Client Components** — 不同执行环境
