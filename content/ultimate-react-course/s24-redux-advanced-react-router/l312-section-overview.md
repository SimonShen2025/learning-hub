## 中文短总结

本节实现应用核心：用 Redux Toolkit 构建购物车，深入 React Router 数据加载特性。涵盖 Redux 建模、Selectors、Thunks、createAsyncThunk、useFetcher 等高级特性。

## 中文长总结

### 学习内容
1. 用 Redux Toolkit 建模 user 和 cart 状态
2. 读取、更新 Redux 状态
3. Redux Selectors（计算派生数据）
4. 购物车完整 CRUD（增、删、改数量）
5. Redux Thunks + createAsyncThunk（异步操作）
6. 地理位置 API 集成
7. React Router useFetcher（无导航获取/更新数据）
8. 无导航的表单提交

### 项目实战
- Fast React Pizza 应用的购物车功能
- 购物车状态管理（添加、删除、修改数量）
- 订单提交流程
- 地理位置自动填充地址

## English Short Summary

Section covers Redux Toolkit for cart/user state, Redux selectors, thunks with createAsyncThunk, geolocation integration, and React Router's useFetcher for data fetching/mutation without navigation.
