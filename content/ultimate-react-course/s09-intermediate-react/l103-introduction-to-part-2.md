## 中文短总结

Part 2 开篇介绍。中级 React 阶段将构建两个项目：usePopcorn（电影评分应用）和 Classy Weather（天气应用）。深入组件组合/复用、React 内部机制、Effects/数据获取、自定义 Hooks、Refs，以及可选的 Class 组件。

## 中文长总结

### Part 2 内容概览
- **usePopcorn 项目**：电影搜索和评分应用，贯穿多个 section
- **Classy Weather 项目**：天气应用（Class 组件版本）

### 核心主题
1. 组件组合与复用性（Composition & Reusability）
2. React 内部工作原理（渲染、Diffing、Fiber 等）
3. Effects 与数据获取（useEffect、fetch、error handling）
4. 自定义 Hooks 与 Refs（useRef、自定义 hooks）
5. [可选] Class 组件（React Hooks 之前的写法）

### 进阶要求
- 需扎实掌握 Part 1 基础
- 开始接触 React 高级概念
