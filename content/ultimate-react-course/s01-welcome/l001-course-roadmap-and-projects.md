## 中文短总结

课程分为四大部分：Part 1 学 React 基础（components、JSX、props、state、forms），包含大量练习；Part 2 深入中级概念（data fetching、useEffect、custom hooks、React 工作原理）；Part 3 进阶主题（reducers、性能优化、React Router、Redux/Redux Toolkit）；Part 4 用行业标准工具（Redux、Tailwind、React Query、Supabase）构建两个完整的专业级应用。

## 中文长总结

本课程从零基础到高级 React 开发者，分为四大部分循序渐进：

**Part 1 — React 基础**：通过多个小项目快速上手，学习 components、JSX、props、state、forms 等核心概念，配合大量 challenges 和 exercises 实战练习。

**Part 2 — 中级 React**：在基础之上学习 data fetching、useEffect hook、custom hooks，并有专门章节讲解 React 底层运作原理（behind the scenes），帮助建立深层理解和自信。

**Part 3 — 高级 React**：深入 reducers、性能优化，用 React Router 构建第一个 SPA，学习 Redux 和现代 Redux Toolkit。

**Part 4 — 实战应用**：将所学知识应用于真实场景，使用 Redux、Tailwind、React Query、Supabase 等工具构建两个完整的专业级应用。

## English Short Summary

The course is structured in 4 parts: Part 1 covers React fundamentals (components, JSX, props, state, forms) with hands-on projects. Part 2 tackles intermediate topics (data fetching, useEffect, custom hooks, React internals). Part 3 advances to reducers, performance optimization, React Router, and Redux. Part 4 builds two professional apps using Redux, Tailwind, React Query, and Supabase.
