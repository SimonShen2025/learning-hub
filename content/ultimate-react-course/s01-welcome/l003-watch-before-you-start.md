## 中文短总结

学习 React 的 8 条建议：不同水平的学生都适合本课程，可调节速度/跳过部分节省时间；必须自己写代码（光看不行）；初期别纠结原理和最佳实践，先让东西跑起来；遇到问题先自己尝试解决，再查 Q&A，最后再提问（附 CodeSandbox 链接）；每个 section 学完后要复习总结，不要急于赶进度；所有操作系统通用；最重要——保持乐趣。

## 中文长总结

### 课程适用性
- 课程面向所有水平，从零基础到想进阶的开发者都适用。可通过调节播放速度、跳过重复练习部分来缩短学习时间（至少可以减半）。

### 学习方法
1. **必须动手写代码**——即使只是照着敲一遍，代码也会进入大脑。光看视频学不到任何 React 技能。
2. **初学阶段别纠结 why/how**——前几个 section 先关注"让它跑起来"，原理和最佳实践后面会逐步展开。
3. **问题解决流程**：先自己尝试 → 搜索 Q&A 区已有答案 → 新提问时附上 CodeSandbox 链接和简短描述。
4. **每个 section 完成后复习**：回顾代码、笔记和项目，甚至自己写些练习代码，确保理解后再继续下一部分。

### 其他
- Mac/Windows/Linux 行为完全一致。
- 保持学习的乐趣，遇到挫折就休息一下再回来。

## English Short Summary

8 tips for taking the course: always code along (never just watch), don't worry about "why" in early sections, solve problems yourself before asking in Q&A (share CodeSandbox links), review and consolidate after each section before moving on, course works identically on all OS, and most importantly — have fun.
