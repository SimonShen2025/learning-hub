## 中文短总结

本节是 Wild Oasis 项目最大的一节。内容：客户端/服务端过滤排序分页、预获取、预订详情、入住/退房、认证授权、暗色模式、仪表盘图表。

## 中文长总结

### 涵盖内容
1. **过滤和排序** — 客户端（Cabins）+ API 端（Bookings）
2. **分页** — 可复用分页组件 + API 端分页
3. **预获取** — React Query prefetchQuery
4. **预订管理** — 单个预订页面、Check-in/Check-out、早餐加购
5. **认证** — Supabase Auth 登录/注册/登出
6. **授权** — 路由保护 + Supabase RLS
7. **用户管理** — 更新密码/头像/姓名
8. **暗色模式** — CSS 变量实现
9. **仪表盘** — 统计卡片、折线图、饼图
10. **收尾** — Error Boundaries、Bug 修复

## English Short Summary

Largest section of Wild Oasis project: client/API-side filtering, sorting, pagination, prefetching, booking management (check-in/out), authentication, authorization (RLS), dark mode with CSS variables, dashboard with statistics and charts.
