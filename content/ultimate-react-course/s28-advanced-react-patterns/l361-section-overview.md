## 中文短总结

高级 React 模式：Render Props、Higher-Order Components (HOC)、Compound Component Pattern。这些模式用于构建高度可复用、灵活的组件。资深 React 工程师日常使用的实践。

## 中文长总结

### 本节覆盖
1. **Render Props 模式** — 通过 render 函数 prop 决定组件渲染什么
2. **Higher-Order Components (HOC)** — 包装组件添加功能
3. **Compound Component 模式** — 多个子组件协同工作（如 `<Modal.Open>` + `<Modal.Window>`）
4. **React Portal** — 在 DOM 层级外渲染 Modal
5. **检测外部点击** — 关闭 Modal
6. **实战应用** — 可复用 Table、确认删除、Context Menu

### 目标
- 提升组件可复用性和灵活性
- 学习专业级代码模式
- 应用到 Wild Oasis 项目中

## English Short Summary

Advanced React patterns: Render Props, Higher-Order Components, Compound Component Pattern. Build reusable, flexible components. Applied to Modal, Table, and Context Menu in the Wild Oasis project.
