## 中文短总结

Part 1 的收尾实践项目 — Eat-'N-Split（AA 制分账应用）。复习所有 React 基础知识，不引入新概念。功能：管理朋友列表、添加新朋友、选择朋友进行分账、计算谁欠谁钱。综合运用 state、props、受控组件、lifting state up、条件渲染等。

## 中文长总结

### 项目功能
- 显示朋友列表（头像、姓名、余额）
- 添加新朋友
- 选择朋友 → 打开分账表单
- 输入总账单、自己付的金额 → 计算对方应付
- 选择谁付账（You / Friend）
- 更新余额

### 复习要点
- 组件拆分和组合
- State 管理（多个 state 协同）
- 受控组件（多个表单输入）
- Lifting State Up
- 条件渲染
- 列表渲染
- children prop
