## 中文短总结

Wild Oasis 客户网站项目概述。为之前 React 课程构建的酒店管理内部系统创建面向客户的预订网站。使用 Next.js App Router + Tailwind CSS + Supabase。覆盖项目规划、样式、图片优化、路由、布局。

## 中文长总结

### 项目定位
- 之前构建了 Wild Oasis 内部管理系统（React + Supabase）
- 现在构建**客户端网站**：
  - 浏览小屋信息
  - 在线预订
  - 管理预订
  - 用户认证

### 技术栈
- **Next.js** — App Router
- **Tailwind CSS** — 样式
- **Supabase** — 后端（复用之前的数据库）
- **NextAuth.js** — 认证
