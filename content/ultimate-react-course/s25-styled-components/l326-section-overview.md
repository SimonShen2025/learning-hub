## 中文长总结

### 项目概述
- **The Wild Oasis** — 小型精品酒店内部管理应用
- 管理小木屋（cabins）、预订（bookings）、客人（guests）
- **不是**面向客人的网站，而是酒店员工使用的后台

### 技术栈
| 库 | 用途 |

| Styled Components | 样式 |
| React Query | 远程状态管理 |
| React Hook Form | 表单处理 |
| React Router | 路由（SPA） |
| Supabase | 后端 + 数据库 |
| Recharts | 图表 |
