## 中文短总结

"Thinking in React"是每个 React 开发者需要从一开始就培养的核心技能。本 section 覆盖：状态管理基础、状态提升（lifting state up）、子→父通信、派生状态（derived state）、children prop、组件可复用性。继续在 Far Away 项目上构建。

## 中文长总结

本 section 涵盖：
- "Thinking in React" 的含义和方法论
- 状态管理的基本原则（何处放 state、谁需要 state）
- **Lifting State Up**：将 state 提升到共同父组件
- **子→父通信**：通过回调函数（child-to-parent communication）
- **Derived State**：从现有 state 计算而非创建新 state
- **children prop**：构建可复用的容器组件
- 完善 Far Away 项目：删除、更新、排序、清空功能
- 将组件拆分到独立文件

## English Short Summary

Core "Thinking in React" skills: state management fundamentals, lifting state up to common parent, child-to-parent communication via callbacks, derived state, and the children prop for reusable containers. Continues building the Far Away project with delete, update, sort, and clear features.
