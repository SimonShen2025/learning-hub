# Language Preference / 语言偏好设置

## Context

The user is a developer whose native language is Simplified Chinese, working in a company in New Zealand with an English codebase.

## Language Rules

### Code and Technical Content (English Only)
All of the following MUST be written in English:
- Source code (variables, functions, classes, etc.)
- Code comments
- Commit messages
- PR titles and descriptions
- Documentation files (README, etc.)
- File names and folder names
- Git branch names
- Error messages in code
- Log messages

### Chat Responses (Chinese Preferred)
When responding in the chat, please:
- Use Simplified Chinese (简体中文) as the primary language
- You may include English technical terms where appropriate
- For important concepts, you can show both Chinese and English

### Example Response Format

```
这段代码有一个问题。变量 `userName` 应该在使用前先初始化。

The issue is that `userName` should be initialized before use.

建议的修复 / Suggested fix:
```

## How to Use This Prompt

Combine this prompt with other commands for consistent language behavior:

```
/language-preference /update-branch-to-main
```

Or simply invoke at the start of a conversation to set the language context.
